## はじめに

研修医・若手医師にとって、毎週のカンファレンス発表は避けて通れない業務です。
症例発表、抄読会、M&Mカンファレンスなど、発表の機会は多岐にわたります。
しかし、日常の診療業務に追われる中で、質の高い発表資料を準備する時間を確保するのは容易ではありません。

このガイドでは、AIを活用して発表資料作成を効率化する方法を紹介します。
症例情報を入力するだけで、標準的なスライド構成、鑑別診断リスト、参考文献の提案まで、AIが自動的にサポートします。

---

## なぜAI支援が必要なのか？

### 研修医の現状

週平均労働時間は56時間22分で、法定労働時間を大幅超過しています。
カンファレンス発表は週1〜2回あります。
資料作成時間は1回あたり2〜4時間かかります。
結果として、睡眠時間を削って資料作成を行っています。

### AI支援の効果

時間短縮として、2〜4時間から30分〜1時間に短縮できます。
質の向上として、標準的な構成で見落としを防げます。
学習効果として、AIの提案から学べます。
ストレス軽減として、「何を書くべきか」の悩みを解消できます。

# ステップ1: カンファレンス発表の標準的な構成を理解する

## 症例カンファレンスの基本構成

質の高い症例発表には、標準的な構成があります。
この構成に沿って資料を作成することで、聴衆が理解しやすく、議論が活発になります。

### 標準的なスライド構成（10-15分発表）

標準的な症例発表は、約10スライド、15分で構成されます。

スライド1はタイトルスライドで、所要時間は30秒です。
症例の概要を一言で伝えることがポイントです。
スライド2は症例サマリーで、所要時間は1分です。
年齢、性別、主訴、診断を含めることがポイントです。
スライド3は現病歴で、所要時間は2分です。
時系列で簡潔に伝えることがポイントです。
スライド4は身体所見で、所要時間は1分です。
重要な所見のみを伝えることがポイントです。
スライド5は検査所見で、所要時間は2分です。
異常値をハイライトすることがポイントです。
スライド6は鑑別診断で、所要時間は2分です。
優先順位付きで伝えることがポイントです。
スライド7は診断根拠で、所要時間は2分です。
なぜこの診断に至ったかを説明することがポイントです。
スライド8は治療方針で、所要時間は2分です。
選択した理由を説明することがポイントです。
スライド9は経過で、所要時間は1分です。
治療後の変化を示すことがポイントです。
スライド10は考察・学び（Take-home message）で、所要時間は2分です。

合計で約10スライド、15分の構成となります。

---

## 各スライドの詳細

### 1. タイトルスライド

目的は、症例の核心を一言で伝えることです。

含めるべき情報として、症例の特徴（年齢・性別）があります。
主訴または診断を含めます。
発表者名・所属を含めます。
発表日を含めます。

良い例:
```
80歳女性、急性腹症で来院
診断: 上腸間膜動脈血栓症

発表者: 山田太郎（内科研修医）
2025年12月13日
```

悪い例:
```
症例発表

山田太郎
```

---

### 2. 症例サマリー

目的は、症例の全体像を1分で理解させることです。

含めるべき情報として、年齢・性別があります。
主訴を含めます。
現病歴を1-2文で簡潔に含めます。
診断を含めます。
転帰を含めます。

テンプレート:
```
【症例サマリー】
- 年齢・性別: 80歳、女性
- 主訴: 急性腹痛
- 現病歴: 来院3時間前から突然の腹痛が出現。嘔吐を伴う。
- 診断: 上腸間膜動脈血栓症
- 転帰: 緊急手術施行、術後経過良好
```

---

### 3. 現病歴

目的は、症状の経過を時系列で理解させることです。

時系列で整理し、発症時刻を明確にします。
重要な症状のみを伝え、詳細すぎないようにします。
関連する既往歴を必要に応じて含めます。

良い例:
```
【現病歴】
- X日 15:00: 突然の腹痛が出現（心窩部→全腹部）
- X日 16:00: 嘔吐2回、下痢なし
- X日 18:00: 症状増悪し救急搬送

【既往歴】
- 心房細動（ワルファリン内服中断中）
- 高血圧、脂質異常症
```

---

### 4. 身体所見

目的は、診断に重要な所見を強調することです。

バイタルサインを必ず記載します。
陽性所見を強調します。
陰性所見も重要で、除外診断に役立ちます。

テンプレート:
```
【バイタルサイン】
- 体温: 37.8°C
- 血圧: 100/60 mmHg
- 脈拍: 110回/分（不整）
- SpO2: 96%（室内気）

【腹部所見】
- 全腹部に圧痛（+）、筋性防御（-）
- 腸蠕動音: 減弱
- 直腸診: 血便（-）
```

---

### 5. 検査所見

目的は、診断に必要な検査結果を提示することです。

異常値をハイライトします。
太字や色付けで強調します。
正常値も記載し、基準範囲を示します。
画像所見は別スライドで提示します。

良い例:
```
【血液検査】
- WBC: 15,000 /μL ↑
- CRP: 8.5 mg/dL ↑
- Lactate: 4.2 mmol/L ↑
- D-dimer: 5.8 μg/mL ↑

【CT所見】
- 上腸間膜動脈に血栓（+）
- 小腸壁の造影不良
- 腹水（+）
```

---

### 6. 鑑別診断

目的は、診断プロセスを明示することです。

優先順位付きで、頻度や重症度を考慮します。
除外した理由も記載します。
3-5個に絞って提示します。

テンプレート:
```
【鑑別診断】
1. 上腸間膜動脈血栓症 ★
   - 心房細動、急性腹痛、Lactate上昇
2. 腸閉塞
   - 腸蠕動音減弱、嘔吐
   → CT で血栓を確認し除外
3. 急性膵炎
   - 腹痛、嘔吐
   → アミラーゼ正常で除外
```

---

### 7. 診断根拠

目的は、なぜこの診断に至ったかを説明することです。

臨床所見と検査所見を統合します。
診断基準があれば提示します。
確定診断の方法を説明します。

例:
```
【診断根拠】
1. 臨床所見
   - 心房細動（血栓リスク）
   - 急性腹痛（腹部所見に乏しい）
   
2. 検査所見
   - Lactate上昇（腸管虚血）
   - CT: 上腸間膜動脈血栓

3. 確定診断
   - 造影CT で上腸間膜動脈血栓を確認
```

---

### 8. 治療方針

目的は、選択した治療とその理由を説明することです。

治療選択肢を提示します。
選択した理由を明確にします。
代替案も考慮します。

テンプレート:
```
【治療選択肢】
1. 保存的治療（抗凝固療法）
2. 血管内治療（血栓溶解療法）
3. 外科的治療（血栓除去術）

【選択した治療】
- 緊急開腹術（血栓除去 + 腸管切除）

【理由】
- 腸管壊死の可能性が高い
- 発症から時間が経過
- 保存的治療では改善困難
```

---

### 9. 経過

目的は、治療後の変化を示すことです。

時系列で整理します。
改善点を強調します。
合併症があれば記載します。

例:
```
【術後経過】
- 術後1日目: ICU管理、バイタル安定
- 術後3日目: 一般病棟へ転棟
- 術後7日目: 経口摂取開始
- 術後14日目: 退院

【合併症】
- なし
```

---

### 10. 考察・学び（Take-home message）

目的は、この症例から何を学んだかを明確にすることです。

1-3個の重要なポイントを提示します。
一般化できる知識を伝えます。
今後の診療に活かせることを示します。

テンプレート:
```
【Take-home message】

1. 心房細動患者の急性腹痛では、
   上腸間膜動脈血栓症を疑う

2. 腹部所見に乏しくても、
   Lactate上昇があれば腸管虚血を考慮

3. 早期診断・早期治療が予後を左右する
```

# ステップ2: AIプロンプトの基本を理解する

## AIプロンプトとは？

AIプロンプトとは、AIに指示を出すための「質問文」や「依頼文」のことです。
プロンプトの質が、AIの出力の質を大きく左右します。

### 良いプロンプトの条件

1. 具体的: 何をしてほしいか明確に
2. 構造化: 情報を整理して提示
3. 文脈を提供: 背景情報を含める
4. 出力形式を指定: どのような形式で出力してほしいか

---

## 基本的なプロンプトテンプレート

### テンプレート1: 症例発表スライド生成

```
あなたは経験豊富な臨床医です。以下の症例情報から、カンファレンス発表用のスライド構成を提案してください。

【症例情報】
- 年齢・性別: [情報を入力]
- 主訴: [情報を入力]
- 現病歴: [情報を入力]
- 身体所見: [情報を入力]
- 検査所見: [情報を入力]
- 診断: [情報を入力]
- 治療: [情報を入力]
- 経過: [情報を入力]

【出力形式】
以下の構成でスライドの内容を提案してください：
1. タイトルスライド
2. 症例サマリー
3. 現病歴
4. 身体所見
5. 検査所見
6. 鑑別診断（優先順位付き）
7. 診断根拠
8. 治療方針
9. 経過
10. Take-home message

各スライドには、箇条書きで具体的な内容を記載してください。
```

### Q1: AIの出力をそのまま使っても大丈夫？

A: 必ず医学的な正確性を確認してください。

- 治療ガイドライン

A: 基本的には無料版で十分です。

- より長い症例（複雑な経過）
- より詳細な鑑別診断
- 文献検索も含めたい場合

A: いいえ。

   - 否定的な所見: 両側浸潤影（PEでは通常なし）、下肢腫脹なし
   - 確定診断の方法: D-dimer、造影CT、下肢静脈エコー
   - 緊急性: 高

7. 誤嚥性肺炎
   - 支持する所見: 高齢、両側浸潤影
   - 否定的な所見: 嚥下障害の既往なし（確認必要）
   - 確定診断の方法: 嚥下評価、胸部CT
   - 緊急性: 中
```

## 鑑別診断の優先順位付け

AIが生成した鑑別診断を、以下の基準で優先順位付けします。

### 優先順位の基準

1. 緊急性（見逃してはいけない疾患）
   - 致死的: 肺塞栓症、急性冠症候群、大動脈解離
   - 緊急治療が必要: 急性心不全、肺炎、敗血症

2. 頻度（Common disease）
   - よくある疾患を優先

3. 所見の一致度
   - 支持する所見が多い疾患を優先

### 優先順位付けのプロンプト

```
上記の鑑別診断を、以下の基準で優先順位付けしてください：

1. 緊急性（見逃してはいけない疾患）
2. 頻度（Common disease）
3. 所見の一致度

最も可能性の高い診断を★で示してください。
```

## 鑑別診断の除外根拠を明確にする

カンファレンスでは、「なぜこの診断を除外したか」を説明することが重要です。

### 除外根拠のプロンプト

```

- どの所見が否定的だったか
- どの検査で除外できたか
- 除外診断の確実性（完全に除外 / 可能性は低い）
```

---

### 出力例

```
【除外した診断とその根拠】

1. COPD急性増悪
   - 除外根拠: 両側肺野に浸潤影あり（COPDでは通常なし）
   - 除外の確実性: 可能性は低い（完全除外ではない）
   - 追加検討: 呼吸機能検査で確認が望ましい

2. 肺塞栓症
   - 除外根拠: 
     - 両側浸潤影あり（PEでは通常なし）
     - 下肢腫脹なし
     - D-dimer測定していない（要確認）
   - 除外の確実性: 可能性は低い
   - 追加検討: D-dimer測定を推奨

3. 間質性肺炎
   - 除外根拠: 急性発症（間質性肺炎は通常慢性）
   - 除外の確実性: 可能性は低い
   - 追加検討: HRCT、KL-6で確認が望ましい
```

## 発表シナリオ生成のプロンプト

```

【スライド構成】
[ここにスライドの内容を貼り付け]

【発表条件】
- 発表時間: 15分
- 聴衆: 内科研修医・指導医
- 発表スタイル: フォーマル

【出力形式】

【スライド1: タイトル】
所要時間: 30秒

所要時間: 1分
シナリオ: [話す内容]

1. 術後患者の呼吸困難では、肺塞栓症を必ず鑑別に挙げる

## 質問のパターンと対処法

### パターン1: 診断プロセスに関する質問

例:
- 「なぜこの検査を選択したのですか?」
- 「他の検査は検討しましたか?」

対処法:
- 検査の感度・特異度を説明
- ガイドラインを引用
- コスト・侵襲性を考慮した理由を説明

---

### パターン2: 治療選択に関する質問

例:
- 「なぜこの治療を選択したのですか?」
- 「他の治療法は検討しましたか?」

対処法:
- 治療選択肢を列挙
- 選択した理由を明確に（エビデンス、患者背景）
- ガイドラインを引用

---

### パターン3: 鑑別診断に関する質問

例:
- 「〇〇という診断は考えましたか?」
- 「なぜ〇〇を除外したのですか?」

対処法:
- 鑑別診断リストに含めていれば、除外理由を説明
- 含めていなければ、「ご指摘ありがとうございます。確かに鑑別に挙げるべきでした」

---

### パターン4: 予後・フォローアップに関する質問

例:
- 「退院後のフォローアップはどうしますか?」
- 「再発リスクはどのくらいですか?」

対処法:
- フォローアップ計画を具体的に説明
- 再発リスクのデータを引用
- 予防策を説明

A11: S1Q3T3パターンの感度は約20-50%、特異度は約80-90%です。感度は低いため、S1Q3T3がなくても肺塞栓症は除外できません。特異度は比較的高いため、認めた場合は肺塞栓症を強く疑います。

Q12: 造影CTで血栓が見つからなかった場合、どうしますか?
A12: 造影CTの感度は約90%以上ですが、末梢の小さな血栓は見逃す可能性があります。臨床的に肺塞栓症を強く疑う場合は、肺換気血流シンチグラフィーや肺動脈造影を検討します。

Q13: 術後VTE予防は行われていましたか?
A13: カルテを確認したところ、術後は早期離床のみで、薬物的予防は行われていませんでした。整形外科手術は高リスクであり、予防的抗凝固療法が推奨されます。今回の症例を教訓に、術後VTE予防の重要性を再認識しました。

A14: 糖尿病と高血圧がありましたが、これらは直接的なVTEのリスク因子ではありません。最も重要なリスク因子は、術後という点です。年齢（65歳）も軽度のリスク因子です。

# 放射線画像診断AIの現在地と未来

## 画像診断AIは「黎明期」を超えた

2020年代前半、「AIが放射線科医を置き換える」という議論が盛んでした。2026年の現実は、それとは異なる形で進化しています。AIは放射線科医を置き換えるのではなく、「AIを使う放射線科医が、使わない放射線科医を置き換える」方向に収束しつつあります。

FDA承認済みのAI医療機器は2026年3月時点で1,000件を超え、そのうち約75%が放射線科領域です。これは放射線画像という標準化されたデータ形式がAI開発に適していることを反映しています。

## 臨床導入が進む3つの領域

### 1. 緊急トリアージ（最も成熟）

脳出血、肺塞栓、大動脈解離などの緊急所見を自動検出し、読影キューの優先順位を変更するシステムが最も実用化されています。

**臨床的インパクト:**
- 緊急所見の報告時間が平均60%短縮
- 夜間・当直帯での見落とし率低下
- ワークフロー効率の改善

代表的な製品として、Aidocの頭部CT出血検出（感度95%以上）やViz.aiの大血管閉塞検出（脳卒中チーム起動を20分以上短縮）があります。

### 2. 定量解析（急成長中）

従来は「主観的」だった画像評価を定量化するAIが増えています。

- **骨密度測定**: 日常的なCTから骨密度を推算し、骨粗鬆症スクリーニング
- **冠動脈石灰化スコア**: 非造影CTからCACスコアを自動算出
- **脳容積測定**: 認知症の経時的変化を定量追跡
- **肺結節測定**: RECIST基準に基づく腫瘍径の自動測定と経時比較

### 3. スクリーニング補助（発展途上）

マンモグラフィのAI読影は欧州で大規模臨床試験が進行中です。スウェーデンのMASSIVE試験（8万人規模）では、AI支援読影が二重読影と同等以上の精度を示し、読影医の負担を半減させる可能性が示されました。

## 導入時の落とし穴

### バイアスの問題

学習データの偏りは深刻です。米国の大規模データセットで学習したモデルが、アジア人患者で精度低下する事例が報告されています。自施設のデータでバリデーションを行うことが不可欠です。

### アラート疲れ

偽陽性が多すぎると、臨床医がアラートを無視するようになります。感度と特異度のバランス、アラートの提示方法の設計が重要です。

### 統合コスト

既存のPACS/RISとの統合には技術的・制度的な課題があります。DICOM標準への準拠は前提ですが、実際の導入には数か月のカスタマイズが必要なケースも多いです。

## AIと放射線科医の協働モデル

最も生産性が高いモデルは「AIファースト・リード」です。

1. **AI前処理**: 画像のセグメンテーション、定量測定、異常候補の抽出
2. **医師確認**: AIの所見を参照しながら最終読影
3. **フィードバック**: 医師の修正がAIの学習データに還元

このモデルでは、放射線科医の役割は「画像を見る人」から「AIの出力を検証し、臨床的文脈を加える人」に変化します。

# 栄養アセスメント・プランニング

## 概要

入院患者の20〜50%が栄養障害を有するとされています。適切な栄養管理は合併症の予防、在院日数の短縮、予後の改善に直結します。このプロンプトは、栄養スクリーニングから栄養管理計画の作成までを体系的に支援します。

## プロンプトテンプレート

```text
あなたはNST（栄養サポートチーム）の経験豊富な医師です。以下の患者情報に基づき、包括的な栄養アセスメントと栄養管理計画を作成してください。

# 患者基本情報
- 年齢: {{年齢}}歳、性別: {{性別}}
- 身長: {{身長}}cm、現体重: {{体重}}kg
- 通常時体重: {{通常体重}}kg（いつの時点: {{時期}}）
- 主病名: {{診断名}}
- 入院理由: {{入院理由}}

# 栄養関連データ
- 血液検査:
  - Alb: {{Alb}}g/dL、Pre-Alb: {{Pre-Alb}}mg/dL
  - CRP: {{CRP}}mg/dL
  - Hb: {{Hb}}g/dL、リンパ球数: {{リンパ球}}
  - BUN: {{BUN}}mg/dL、Cr: {{Cr}}mg/dL
  - Na: {{Na}}mEq/L、K: {{K}}mEq/L
  - 血糖: {{血糖}}mg/dL、HbA1c: {{HbA1c}}%
- 身体所見: {{浮腫の有無、筋肉量、皮下脂肪}}
- 消化管機能: {{経口摂取の可否、嚥下機能、腸蠕動}}
- 現在の栄養摂取: {{食事量、輸液内容}}

# 評価してほしいこと
1. **栄養スクリーニング**: NRS-2002、MNA-SF、GLIM基準での評価
2. **必要エネルギー量**: Harris-Benedictの式＋活動係数＋ストレス係数
3. **必要蛋白量**: 疾患・状態に応じた蛋白必要量（g/kg/日）
4. **栄養投与経路**: 経口/経腸/静脈の選択と根拠
5. **具体的な栄養処方**: 製品名、投与量、投与速度、スケジュール
6. **モニタリング計画**: 再評価の時期と項目
7. **リフィーディング症候群のリスク**: 評価と予防策

# 出力形式
- 計算過程を示してください
- 推奨する栄養製品は一般名と具体的な製品名の両方を記載
- 注意：最終的な栄養管理計画は、NST回診で多職種で検討してください
```

## 使い方

1. 血液検査データはできるだけ最新のものを入力
2. 炎症がある場合、Albは栄養指標として信頼性が低下する（CRPと合わせて評価）
3. 通常時体重からの変化率が重要な指標
4. 消化管機能の情報は栄養経路選択の根拠になる

## 活用のポイント

- GLIM基準（2019年）は国際的な栄養障害診断基準として広く採用
- リフィーディング症候群のリスク評価は、長期絶食・低栄養患者で特に重要
- 経腸栄養は可能な限り静脈栄養よりも優先する（"If the gut works, use it"）

# インフォームドコンセント文書生成

## 概要

インフォームドコンセントは患者の自己決定権を尊重する医療の基本です。しかし、医学的に正確でありながら患者に理解しやすい説明文書を作成するのは時間のかかる作業です。このプロンプトは、手術・検査・治療の説明文書の下書きを生成します。

## 安全に関する重要な注意事項

生成された文書は必ず担当医が内容を確認・修正してください。施設ごとの同意書フォーマットに合わせた調整が必要です。法的文書としての有効性は、AIの出力ではなく、医師と患者間の実際のコミュニケーションに基づきます。

## プロンプトテンプレート

```text
あなたは患者コミュニケーションに長けた経験豊富な外科医です。以下の手術/検査/治療について、患者に分かりやすいインフォームドコンセント説明文書を作成してください。

# 基本情報
- 処置名: {{処置名}}（例: 腹腔鏡下胆嚢摘出術）
- 対象患者: {{年齢}}歳、{{性別}}
- 診断名: {{診断名}}
- 緊急度: {{予定/準緊急/緊急}}

# 説明文書に含めてほしい内容
1. **病状の説明**: なぜこの処置が必要か（医学用語を使わず平易に）
2. **処置の内容**: 具体的に何をするか（麻酔方法、所要時間含む）
3. **期待される効果**: 処置によって何が改善するか
4. **リスクと合併症**:
   - 一般的なリスク（頻度：%で記載）
   - 稀だが重大なリスク
   - この患者に特有のリスク（既往歴に基づく）
5. **代替治療法**: 処置を行わない場合の経過、他の治療選択肢
6. **術後の経過**: 入院期間、日常生活への復帰時期、フォローアップ
7. **費用**: 概算（保険適用/自費）

# 文書のトーン
- 中学生が理解できるレベルの日本語
- 恐怖を煽りすぎず、かつリスクを過小評価しない
- 「〜かもしれません」「〜の可能性があります」など適切な表現
- 患者が質問しやすい雰囲気を作る文末

# 注意事項
- 生成された文書は担当医が必ず確認・修正してください
- 施設の同意書フォーマットに合わせて調整してください
```

## 使い方

1. 処置名は正式名称と一般的な呼び方の両方を入力

3. 生成後は必ず専門医が内容を確認

- 外国人患者向けに英語版の生成も可能

【カテゴリー1: 診断プロセスに関する質問】

Q1: Wells scoreは計算しましたか? 何点でしたか?
A1: はい、Wells scoreは7点でした。内訳は、DVTの臨床症状で3点、PEが最も疑わしいで3点、心拍数>100で1.5点です。7点以上は「PEの可能性が高い」と判定されます。

Q2: D-dimerのカットオフ値はどのように考えましたか?
A2: 一般的なカットオフ値は0.5μg/mL以下ですが、本症例では8.5μg/mLと著明に上昇しており、VTEを強く示唆しました。D-dimerは特異度が低いため、画像検査での確認が必須です。

Q3: 造影CTの前に、心エコーは施行しましたか?
A3: 本症例では、バイタルが比較的安定していたため、造影CTを優先しました。心エコーは右室負荷の評価に有用ですが、確定診断には造影CTが必要です。

【カテゴリー2: 治療選択に関する質問】

Q4: なぜヘパリンからDOACに切り替えたのですか?
A4: 急性期はヘパリンで迅速な抗凝固を行い、安定後はDOACに切り替えました。DOACはINRモニタリングが不要で、外来管理が容易です。EINSTEIN-PE試験でDOACの有効性が示されています（N Engl J Med 2012）。

Q5: 血栓溶解療法は検討しましたか?
A5: 本症例では、バイタルが安定しており、大量肺塞栓症の基準を満たさなかったため、血栓溶解療法は行いませんでした。血栓溶解療法の適応は、ショックや持続的な低血圧がある場合です。

Q6: DOACの種類は何を選択しましたか? その理由は?
A6: リバーロキサバンを選択しました。理由は、肺塞栓症での有効性が証明されており（EINSTEIN-PE試験）、1日2回の内服で済むためです。腎機能が保たれていたため、用量調整も不要でした。

【カテゴリー3: 鑑別診断に関する質問】

Q7: 急性冠症候群をどのように除外しましたか?
A7: 胸痛がなく、心電図でもST変化を認めませんでした。トロポニンも陰性でした。S1Q3T3パターンは肺塞栓症に特徴的ですが、急性冠症候群でも見られることがあるため、注意が必要です。

Q8: 心不全との鑑別はどのように行いましたか?
A8: BNPは150pg/mLと軽度上昇のみでした。心不全であれば、より高値が予想されます。心エコーでも明らかな左室機能低下を認めませんでした。右室負荷は認めました。

【カテゴリー4: 予後・フォローアップに関する質問】

Q9: 抗凝固療法の期間はどのくらいですか?
A9: ガイドラインでは、誘因のあるVTE（本症例では術後）の場合、3ヶ月間の抗凝固療法が推奨されています。本症例も3ヶ月間DOACを継続する予定です。その後、再発リスクを評価して、継続の可否を判断します。

Q10: 退院後のフォローアップはどのように行いますか?
A10: 退院後1ヶ月、3ヶ月で外来フォローを行います。DOACのアドヒアランス、出血合併症の有無、下肢腫脹の改善を確認します。3ヶ月後に下肢静脈エコーで血栓の残存を評価する予定です。
