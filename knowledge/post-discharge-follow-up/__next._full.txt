# 医療AIワークフローガイド：退院後フォローアップ計画作成

## イントロダクション

### 目的とAIの役割

退院後フォローアップ計画の作成は、患者の再入院防止、治療アドヒアランスの向上、そしてスムーズな在宅復帰を実現するために極めて重要です。しかし、多忙な臨床現場において、個々の患者の複雑な病状、生活環境、社会的要因をすべて考慮に入れた個別化された計画を迅速に作成することは大きな負担となっています。

AIは、このプロセスにおいて以下の役割を果たします。

1.  情報抽出と要約: 膨大な診療記録（退院サマリー、看護記録、リハビリ記録など）から、計画作成に必要な必須情報（病名、治療内容、内服薬、生活制限、社会的リスク）を迅速かつ正確に抽出・要約します。
2.  個別化の支援: 標準的な疾患別ガイドラインに基づきながら、抽出された患者固有の個別要因（独居、認知機能、介護度、家族構成など）を考慮に入れ、計画をカスタマイズするための示唆を提供します。
3.  多職種連携の効率化: 作成した計画を、医師、看護師、薬剤師、ケアマネジャーなどの多職種間で共有しやすい形式に整理し、連携の円滑化を支援します。

本ガイドでは、AIを活用して、包括的かつ個別化された退院後フォローアップ計画を効率的に作成するワークフローを、基本から応用まで段階的に解説します。

---

## 基本編：情報抽出と標準計画の作成

基本編では、診療記録から計画の土台となる情報を抽出し、疾患に基づいた標準的なフォローアップ計画のドラフトを作成します。

### ステップ 1: 必須情報の抽出と要約（計画の土台作り）

#### 具体的なステップ
1.  患者の退院サマリー、退院時処方箋、および看護サマリーのテキストデータをAIに入力します。
2.  AIに対し、計画作成に不可欠な要素（主要病名、治療内容、退院時状態、内服薬、生活上の制限事項）を構造化された形式（箇条書きや表）で抽出・要約するよう指示します。

#### プロンプト例
```
以下の診療記録テキストから、退院後フォローアップ計画の作成に必要な情報を抽出してください。
1. 主要病名（ICD-10コードもあれば）
2. 今回の入院で行われた主要な治療・手術
3. 退院時内服薬（薬剤名、用量、回数）
4. 退院時のADL/認知機能の状態
5. 今後の生活上の制限事項（食事、運動、入浴など）

[ここに退院サマリー、処方箋、看護サマリーのテキストを貼り付け]
```

#### AIの出力例
```markdown
退院後フォローアップ計画 必須情報要約

1.  主要病名: 慢性心不全（I50.9）、高血圧症（I10）
2.  主要治療: 心不全増悪に対する利尿薬調整、心臓リハビリテーション
3.  退院時内服薬:
    *   エナラプリル 5mg 1T 1日1回朝食後
    *   フロセミド 20mg 1T 1日2回朝夕食後
    *   アスピリン 100mg 1T 1日1回朝食後
4.  ADL/認知機能: ADL自立（IADL一部介助）、MMSE 25点（軽度認知機能低下あり）
5.  生活制限: 塩分制限（6g/日未満）、中等度までの有酸素運動（医師の許可を得て開始）
```

#### まとめ
AIによる情報抽出は、計画作成者が膨大な記録を読み込む時間を大幅に短縮し、情報の見落としを防ぐための基盤となります。特に、多剤併用患者の服薬情報や、複数の制限事項を整理する際に有効です。

### ステップ 2: 標準的なフォローアップ項目のリストアップ（疾患別テンプレート適用）

#### 具体的なステップ
1.  ステップ1で抽出された主要病名と治療内容をAIに提示します。
2.  AIに対し、その疾患の標準的な診療ガイドラインに基づき、退院後1ヶ月、3ヶ月、6ヶ月の受診時期、検査項目、生活指導のポイントを提案するよう指示します。

#### プロンプト例
```
主要病名「慢性心不全」と「高血圧症」に基づき、標準的な退院後フォローアップ計画のドラフトを作成してください。
以下の項目を含めてください。
1.  退院後1ヶ月以内の受診推奨時期と目的
2.  推奨される検査項目（血液検査、画像検査など）
3.  患者に指導すべき具体的な症状モニタリングのポイント
4.  生活指導（食事、運動、服薬）の最重要事項
```

#### AIの出力例
```markdown
標準フォローアップ計画ドラフト（慢性心不全・高血圧症）

| 期間 | 受診推奨時期 | 検査項目 | 指導・モニタリングのポイント |
| :--- | :--- | :--- | :--- |
| 1ヶ月 | 退院後7日〜14日 | 血液検査（BNP/NT-proBNP、腎機能、電解質）、胸部X線 | 体重の毎日の測定（2日で2kg以上の増加は要連絡）、下肢の浮腫、息切れの悪化 |
| 3ヶ月 | 1ヶ月後の受診から4〜6週間後 | 血液検査、心電図 | 血圧の自己測定（目標値130/80mmHg未満）、服薬アドヒアランスの確認 |
| 6ヶ月 | 3ヶ月後の受診から3ヶ月後 | 心エコー検査、血液検査 | 心臓リハビリテーションの継続状況、生活習慣の定着度 |

生活指導の最重要事項:
*   塩分制限の徹底（6g/日未満）
*   利尿薬の確実な服用と、脱水・低血圧症状の注意
*   心不全悪化のサイン（息切れ、浮腫、体重増加）を認識し、早期に医療機関に連絡すること
```

#### まとめ
標準計画のドラフト作成により、ガイドライン遵守の基盤が確立されます。このドラフトは、次の実践編で患者の個別要因に合わせて調整・補強されます。

---

## 実践編：個別化と多職種連携への応用

実践編では、患者の個別要因を深く掘り下げ、計画をカスタマイズするとともに、多職種連携のための実用的な情報整理を行います。

### ステップ 3: 個別リスク要因の評価と計画への組み込み（計画の個別化）

#### 具体的なステップ
1.  ステップ1で抽出された情報に加え、患者の生活環境（独居、家族のサポート状況、自宅の構造、経済状況）や認知機能に関する詳細な情報をAIに入力します。
2.  AIに対し、これらの個別要因が標準計画の実行に与えるリスクを評価させ、そのリスクを軽減するための具体的な支援策を計画に追記するよう指示します。

#### プロンプト例
```
以下の個別要因を考慮し、ステップ2で作成した標準フォローアップ計画を修正・補強してください。
個別要因：
- 独居、近隣に家族なし
- 軽度の認知機能低下（MMSE 25点）
- 経済的な問題はなし
- 自宅は階段あり

特に「服薬管理」と「緊急時の対応」について、具体的な支援策（例：訪問看護、服薬カレンダー、緊急連絡先の設定）を追記し、計画を完成させてください。
```

#### AIの出力例
```markdown
個別化フォローアップ計画（抜粋：リスク対応策）

| リスク要因 | 計画への組み込み（支援策） | 担当職種 |
| :--- | :--- | :--- |
| 独居・認知機能低下 | 服薬管理支援: 訪問看護による週2回の服薬確認とセット、服薬カレンダーの使用を導入。 | 訪問看護師、薬剤師 |
| 緊急時の対応 | 緊急連絡体制の構築: 地域の見守りサービスと連携。緊急連絡先リスト（かかりつけ医、訪問看護、家族）を冷蔵庫に貼付。 | ケアマネジャー、訪問看護師 |
| 自宅の階段 | リハビリ調整: 階段昇降時の息切れに注意し、無理のない範囲での運動を指導。必要に応じて手すりの設置を検討（ケアマネに相談）。 | 理学療法士、ケアマネジャー |

受診推奨時期: 認知機能低下のため、退院後7日以内に外来受診を設定し、計画の理解度を確認する。
```

#### まとめ
個別要因の評価は、計画の実効性を担保するために不可欠です。AIは、複雑に絡み合う社会的・身体的要因を整理し、多職種連携の必要性を明確化する上で強力なツールとなります。

### ステップ 4: 多職種連携のための情報整理（アクションリスト化）

#### 具体的なステップ
1.  完成した個別化フォローアップ計画全体をAIに入力します。
2.  AIに対し、計画に記載されているタスクを、職種別（医師、病棟看護師、訪問看護師、ケアマネジャー、薬剤師、リハビリスタッフ）に具体的なアクションリストとして整理するよう指示します。

#### プロンプト例
```
作成した個別化フォローアップ計画全体を読み込み、以下の職種が退院までに、または退院後に実行すべき具体的なアクションを、それぞれ箇条書きでリストアップしてください。
対象職種：病棟看護師、訪問看護師、ケアマネジャー、薬剤師
```

#### AIの出力例
```markdown
多職種連携アクションリスト

| 職種 | 退院までに実行すべきアクション | 退院後に実行すべきアクション |
| :--- | :--- | :--- |
| 病棟看護師 | ・退院時指導の実施（心不全悪化サイン、体重測定方法） ・訪問看護ステーションへの情報提供（特に認知機能低下と服薬管理の必要性） | - |
| 訪問看護師 | ・初回訪問日の調整と情報共有 | ・週2回の服薬確認とセットアップ ・バイタルサイン、浮腫、息切れのモニタリングと医師への報告 ・緊急連絡体制の機能確認 |
| ケアマネジャー | ・訪問看護、見守りサービスの契約調整 ・自宅の環境整備（手すり設置の検討） | ・サービス利用状況のモニタリングと調整 ・経済的支援の必要性の再評価 |
| 薬剤師 | ・退院時服薬指導（利尿薬の注意点、相互作用） ・服薬カレンダーの使用指導 | ・訪問看護師と連携した服薬アドヒアランスの確認 |
```

#### まとめ
職種別のアクションリストは、タスクの明確化と責任の所在をはっきりさせ、退院支援チーム全体の連携ミスを防ぎます。

## 応用編：患者向け説明資料の作成とリスク予測

応用編では、計画を患者自身が理解しやすい形に変換し、さらにAIによる再入院リスクの予測を組み込むことで、計画の質を最大化します。

### ステップ 5: 患者・家族向け説明資料の作成（平易化と行動変容支援）

#### 具体的なステップ
1.  完成した個別化フォローアップ計画全体をAIに入力します。
2.  AIに対し、計画の内容を患者本人と家族が理解できる平易な言葉に変換し、行動変容を促すような具体的な説明資料（Q&A形式やチェックリスト形式）を作成するよう指示します。

#### プロンプト例
```
作成したフォローアップ計画を、中学生でも理解できる平易な言葉に変換し、患者向けの「退院後の生活チェックリスト」を作成してください。
以下の項目を含めてください。
1.  毎日チェックすること（体重、足のむくみ、息切れ）
2.  絶対に守ること（薬の飲み方、塩分制限）
3.  すぐに病院に連絡すること（具体的な症状3つ）
```

#### AIの出力例
```markdown
【退院後の安心生活チェックリスト】

| 項目 | 内容 | 実行状況 |
| :--- | :--- | :--- |
| 毎日 | 体重を測る: 毎日同じ時間に測り、2日で2kg以上増えていないか確認しましょう。 | □ |
| 毎日 | 足のむくみをチェック: 靴下の跡が深く残っていないか、足が重くないか確認しましょう。 | □ |
| 毎日 | 息切れをチェック: 少し動いただけで息が切れないか確認しましょう。 | □ |
| 重要 | お薬を忘れずに飲む: 訪問看護師さんと一緒にセットしたお薬を、必ず飲みましょう。 | □ |
| 重要 | 塩分を控える: 味が濃いもの、漬物、インスタント食品は控えめにしましょう。 | □ |

【すぐに病院に連絡するサイン】
1.  急に息苦しくなって、横になれないとき
2.  体重が2日で2kg以上増えたとき
3.  熱が出て、咳がひどくなったとき
```

#### まとめ
患者向け資料の作成は、医療リテラシーの格差を埋め、患者の自己管理能力を高める上で極めて重要です。AIは、専門用語を避け、行動に直結する分かりやすい言葉を選ぶ作業を支援します。

### ステップ 6: 再入院リスク予測と計画の最終調整（予後改善への貢献）

#### 具体的なステップ
1.  患者の全診療データ（既往歴、検査値の推移、入院中の合併症、社会的リスク要因）をAIに入力します。
2.  AIに対し、これらのデータに基づき、退院後30日以内の再入院リスクを評価させます（例：低、中、高）。
3.  リスクが高いと評価された場合、AIにリスク低減のための追加介入策（例：退院後早期の電話フォロー、自宅訪問回数の増加、特定の検査の早期実施）を提案させ、計画に最終調整を加えます。

#### プロンプト例
```
以下の患者データに基づき、退院後30日以内の再入院リスクを「低」「中」「高」で評価してください。
また、リスク評価に基づき、現在のフォローアップ計画に追加すべき介入策を提案してください。

[患者データ：年齢、既往歴、入院中のBNP最高値、退院時BNP、腎機能（eGFR）、独居の有無、認知機能スコア]
```

#### AIの出力例
```markdown
再入院リスク評価と最終調整

*   AIによるリスク評価: 中
    *   根拠: 慢性心不全の既往があり、退院時BNP値は改善傾向にあるものの、独居かつ軽度の認知機能低下があるため、服薬アドヒアランス低下による再増悪リスクが残る。
*   追加介入策の提案:
    1.  退院後48時間以内の電話フォロー: 病棟看護師または外来看護師による体調確認と服薬確認を実施。
    2.  訪問看護の頻度調整: 最初の2週間は週3回に増回し、服薬管理と生活環境の安定化を優先する。
    3.  早期検査: 退院後7日目に外来でBNPと腎機能の血液検査を指示。

最終計画: 上記の追加介入策をステップ3の個別化計画に組み込み、最終版とする。
```

#### まとめ
AIによる再入院リスク予測は、限られた医療資源を最も必要とする患者に集中させるためのトリアージを可能にします。これにより、計画の予防的効果が最大化され、医療の質と効率が向上します。

# 放射線画像診断AIの現在地と未来

## 画像診断AIは「黎明期」を超えた

2020年代前半、「AIが放射線科医を置き換える」という議論が盛んでした。2026年の現実は、それとは異なる形で進化しています。AIは放射線科医を置き換えるのではなく、「AIを使う放射線科医が、使わない放射線科医を置き換える」方向に収束しつつあります。

FDA承認済みのAI医療機器は2026年3月時点で1,000件を超え、そのうち約75%が放射線科領域です。これは放射線画像という標準化されたデータ形式がAI開発に適していることを反映しています。

## 臨床導入が進む3つの領域

### 1. 緊急トリアージ（最も成熟）

脳出血、肺塞栓、大動脈解離などの緊急所見を自動検出し、読影キューの優先順位を変更するシステムが最も実用化されています。

**臨床的インパクト:**
- 緊急所見の報告時間が平均60%短縮
- 夜間・当直帯での見落とし率低下
- ワークフロー効率の改善

代表的な製品として、Aidocの頭部CT出血検出（感度95%以上）やViz.aiの大血管閉塞検出（脳卒中チーム起動を20分以上短縮）があります。

### 2. 定量解析（急成長中）

従来は「主観的」だった画像評価を定量化するAIが増えています。

- **骨密度測定**: 日常的なCTから骨密度を推算し、骨粗鬆症スクリーニング
- **冠動脈石灰化スコア**: 非造影CTからCACスコアを自動算出
- **脳容積測定**: 認知症の経時的変化を定量追跡
- **肺結節測定**: RECIST基準に基づく腫瘍径の自動測定と経時比較

### 3. スクリーニング補助（発展途上）

マンモグラフィのAI読影は欧州で大規模臨床試験が進行中です。スウェーデンのMASSIVE試験（8万人規模）では、AI支援読影が二重読影と同等以上の精度を示し、読影医の負担を半減させる可能性が示されました。

## 導入時の落とし穴

### バイアスの問題

学習データの偏りは深刻です。米国の大規模データセットで学習したモデルが、アジア人患者で精度低下する事例が報告されています。自施設のデータでバリデーションを行うことが不可欠です。

### アラート疲れ

偽陽性が多すぎると、臨床医がアラートを無視するようになります。感度と特異度のバランス、アラートの提示方法の設計が重要です。

### 統合コスト

既存のPACS/RISとの統合には技術的・制度的な課題があります。DICOM標準への準拠は前提ですが、実際の導入には数か月のカスタマイズが必要なケースも多いです。

## AIと放射線科医の協働モデル

最も生産性が高いモデルは「AIファースト・リード」です。

1. **AI前処理**: 画像のセグメンテーション、定量測定、異常候補の抽出
2. **医師確認**: AIの所見を参照しながら最終読影
3. **フィードバック**: 医師の修正がAIの学習データに還元

このモデルでは、放射線科医の役割は「画像を見る人」から「AIの出力を検証し、臨床的文脈を加える人」に変化します。

## 今後の展望

- **マルチモーダルAI**: 画像＋電子カルテ＋検査データを統合した診断支援
- **術前シミュレーション**: 3D再構成とAI解析の組み合わせによる手術計画
- **リアルタイム超音波ガイド**: 超音波プローブを握る非専門医をAIがリアルタイム支援

画像診断AIは「使うかどうか」ではなく「どう使うか」のフェーズに入っています。臨床医としては、AIの限界を理解したうえで、自施設のワークフローに最適な形で導入することが重要です。
