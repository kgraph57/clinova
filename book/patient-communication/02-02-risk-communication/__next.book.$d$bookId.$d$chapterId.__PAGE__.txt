## リスクコミュニケーションの重要性

医療におけるリスクコミュニケーションとは、治療や検査に伴うリスクを、患者が正しく理解し、適切な意思決定ができるように伝える技術である。

しかし、リスクを「正確に」伝えることと「わかりやすく」伝えることは、しばしば矛盾する。

### 数字の理解度の現実

2023年の *Annals of Internal Medicine* に掲載された研究では、成人の数値リテラシー（ニューメラシー）の実態が明らかになった。

- **20%を5人に1人と換算できない成人**: 28%
- **「10万人中100人」と「0.1%」が同じと理解できない成人**: 43%
- **相対リスクと絶対リスクの違いを説明できない成人**: 76%
- **「生存率95%」と「死亡率5%」が同じ意味だと理解している成人**: 54%

<Callout type="warning" title="数字の伝え方は「数字そのもの」より重要">
数値リテラシーの低い患者は人口の30〜50%を占める。リスクの数字を正確に伝えても、フォーマットが適切でなければ理解されない。「何を伝えるか」と同時に「どう伝えるか」が決定的に重要である。
</Callout>

---

## リスク表現の4つのフォーマット

### 1. 自然頻度（Natural Frequency）

最も理解されやすい形式。「1,000人中○人」のように、具体的な人数で表現する。

```
推奨: 「この手術を受けた1,000人のうち、30人に合併症が起きました」
非推奨: 「合併症のリスクは3%です」
```

### 2. パーセンテージ

数値リテラシーが高い患者には適切だが、低い患者には混乱を招く。

```
注意: 「50%の確率で改善します」
→ 患者の解釈: 「半分しか治らないなら受けたくない」
→ 意図: 「2人に1人は症状が改善する」
```

### 3. 言語的表現

「まれに」「ときどき」などの言葉は、受け手によって解釈が大きく異なる。

| 医師の意図 | 実際のリスク | 患者が想像するリスク |
|-----------|-------------|-------------------|
| 非常にまれ | 0.01%未満 | 1〜5% |
| まれに | 0.1〜1% | 5〜10% |
| ときどき | 1〜10% | 20〜30% |
| よくある | 10%以上 | 50%以上 |

<Warning title="言語的リスク表現の危険性">
「まれに起こる合併症」と説明した場合、医師は1%未満を想定しているが、患者は5〜10%と解釈する可能性がある。言語的表現は必ず数字と併用すること。
</Warning>

### 4. 視覚的表現（アイコンアレイ）

100人のアイコンを並べ、該当する人数を色で示す方法。最も直感的に理解できる。

```
合併症リスク3%の場合:
●●● ○○○○○○○
○○○○○○○○○○
○○○○○○○○○○
○○○○○○○○○○
○○○○○○○○○○
○○○○○○○○○○
○○○○○○○○○○
○○○○○○○○○○
○○○○○○○○○○
○○○○○○○○○○
● = 合併症あり（3人）  ○ = 合併症なし（97人）
```

### ポジティブフレーム vs ネガティブフレーム

同じリスク情報でも、表現の仕方（フレーム）によって患者の意思決定が変わる。これを**フレーミング効果**と呼ぶ。

**手術**: 腹腔鏡下胆嚢摘出術
**リスクデータ**:
- 術後出血: 0.5%

## 相対リスク vs 絶対リスク

メディアや論文でよく使われる「相対リスク」は、患者を大きく惑わせる。

### 例：新薬の効果を伝える場面

ある新薬が心臓発作のリスクを「50%減少させる」と報道された場合を考える。

| 表現方式 | 伝え方 | 印象 |
|---------|--------|------|
| 相対リスク減少 | 「心臓発作のリスクが50%減少」 | すごく効く薬 |
| 絶対リスク減少 | 「心臓発作のリスクが2%から1%に減少」 | 100人中1人が助かる |
| NNT（治療必要数） | 「100人が飲んで1人の心臓発作を防ぐ」 | 99人には直接効果なし |

<Callout type="info" title="NNT（Number Needed to Treat）の活用">
NNTは「1人の患者を助けるために何人を治療する必要があるか」を示す。相対リスク減少50%（印象的）が、NNT=100（100人治療して1人が助かる）であれば、印象は大きく変わる。リスクコミュニケーションでは、相対値と絶対値の両方を必ず提示する。
</Callout>

<PromptTemplate title="相対リスク→絶対リスク変換プロンプト">
以下の治療効果データを、患者が理解しやすい形式に変換してください。

**データ**:
- 疾患: [例: 心房細動患者の脳卒中予防]
- 薬剤: [例: DOAC（直接作用型経口抗凝固薬）]
- 脳卒中リスク（年間）: 無治療群5%、治療群2%

**出力する形式**:
1. 相対リスク減少（%）
2. 絶対リスク減少（%）
3. NNT（治療必要数）
4. 1,000人の自然頻度表現
5. アイコンアレイ（100人版）
6. 「つまり、こういうことです」と一言で伝える要約文
7. 出血リスク（NNH）も併記して、利益とリスクのバランスを示す
</PromptTemplate>

## リスク比較の「アンカー」

患者がリスクの大きさを判断するために、比較対象（アンカー）を提供することが有効である。

### 日常リスクとの比較表

| リスク | 年間発生率 |
|--------|----------|
| 交通事故で死亡 | 0.004%（10万人中4人） |
| 食中毒にかかる | 0.02% |
| 階段で転倒して入院 | 0.1% |
| インフルエンザにかかる | 5〜10% |
| 風邪をひく | 30〜50% |

<PromptTemplate title="リスクアンカーの生成">
以下の医療リスクについて、患者が大きさを実感できる日常的なリスクとの比較を作成してください。

**医療リスク**: 全身麻酔による死亡リスク（0.001%）

**比較の条件**:
1. 日常生活のリスクとの比較（3つ以上）
2. 同じ数値を異なる表現で3パターン（自然頻度、パーセント、言語的表現）
3. 「これくらいの確率です」と一言で伝えるアナロジー
4. リスクが高くなる要因（年齢、併存疾患など）の注記
</PromptTemplate>

## ヘルスリテラシー別のリスクコミュニケーション

### レベル別アプローチ

| リテラシーレベル | 推奨フォーマット | 避けるべきこと |
|----------------|----------------|--------------|
| 低 | アイコンアレイ＋簡単な言葉 | パーセンテージ、相対リスク |
| 中 | 自然頻度＋バランスフレーム | 相対リスクのみの提示 |
| 高 | 数値データ＋NNT/NNH＋出典 | 過度な簡略化 |

<PromptTemplate title="リテラシー別リスク説明の自動生成">
以下のリスク情報を、ヘルスリテラシーの3レベルに合わせて書き分けてください。

**リスク情報**: 大腸内視鏡検査の穿孔リスク（0.05%）

**レベル1（低リテラシー）**:
- アイコンアレイを使用
- 「めったに起こらない」「10,000人中5人」
- 起こった場合どうするかを明記

**レベル2（中リテラシー）**:
- 自然頻度とパーセンテージの両方
- バランスフレーム
- 日常リスクとの比較1つ

**レベル3（高リテラシー）**:
- 正確なデータと出典
- リスク因子の列挙
- 検査を受けない場合のリスクとの比較
</PromptTemplate>

## 不確実性の伝え方

医療においては、リスクの数字自体に不確実性がある。この「不確実性の不確実性」をどう伝えるかも重要である。

### 不確実性を正直に伝える表現

```
「この合併症の確率は、研究によって1〜5%と幅があります。
 当院では過去3年間で約2%の発生率でした。
 ただし、○○さんの場合は[糖尿病の影響で]少し高くなる可能性があります。
 正確な数字をお伝えすることは難しいですが、
 注意深く対応する体制を整えています。」
```

<Callout type="info" title="「わからない」と言える勇気">
リスクの数字に不確実性がある場合、「正確にはわかりませんが」と正直に伝えることが、むしろ信頼を高める。過度に精密な数字を提示することは、偽りの確実性を与えてしまう。
</Callout>
