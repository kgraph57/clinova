## 小児予防接種 — 最も複雑なスケジュール管理

日本の小児予防接種スケジュールは世界でも有数の複雑さを持つ。定期接種だけで10種類以上のワクチンがあり、生後2ヶ月から接種が始まる。それぞれのワクチンに推奨接種時期、最低接種間隔、接種回数が定められており、これを個々の児の状況に合わせて最適化する必要がある。

<StatHighlight value="20回以上" label="2歳までの接種回数" description="定期接種だけで、2歳までに20回以上の接種が必要。任意接種を含めるとさらに増える" />

### 定期接種ワクチン一覧（2025年度時点）

| ワクチン | 標準的な接種回数 | 開始月齢 |
|---------|--------------|---------|
| B型肝炎 | 3回 | 生後2ヶ月 |
| ロタウイルス | 2回 or 3回 | 生後2ヶ月 |
| ヒブ（Hib） | 4回 | 生後2ヶ月 |
| 小児用肺炎球菌（PCV） | 4回 | 生後2ヶ月 |
| 4種混合（DPT-IPV） | 4回 | 生後2ヶ月 |
| BCG | 1回 | 生後5ヶ月 |
| MR（麻疹・風疹） | 2回 | 1歳 |
| 水痘 | 2回 | 1歳 |
| 日本脳炎 | 4回 | 3歳 |
| HPV | 2-3回 | 小学6年生 |

<Warning>
予防接種スケジュールは毎年のように変更がある。2024年4月からは5種混合（4種混合+Hib）が定期接種化されるなど、最新情報の確認が不可欠である。AIに入力する際は、参照すべきガイドラインの年度を明記すること。
</Warning>

## 臨床シナリオ1：接種遅れのキャッチアップ

### 場面設定

8ヶ月、男児。生後4ヶ月以降の接種が滞っている。引っ越しのため受診が途絶えていた。

接種歴：
- 生後2ヶ月: ロタ1回目、ヒブ1回目、PCV1回目、B肝1回目、4種混合1回目
- 生後3ヶ月: ロタ2回目、ヒブ2回目、PCV2回目、B肝2回目、4種混合2回目
- 以降未接種

<StepFlow>
  <StepFlowStep number={1} title="未接種ワクチンの確認">
    ヒブ3回目、PCV3回目、4種混合3回目、B肝3回目、BCG（5-8ヶ月推奨だが1歳未満まで定期）がすべて未接種。ロタは週数的に3回目（5価の場合）の接種期限を確認。
  </StepFlowStep>
  <StepFlowStep number={2} title="優先順位の決定">
    BCGは1歳の誕生日前日までが定期接種期間のため、急ぐ必要がある。ロタウイルスワクチンは月齢制限（5価: 32週0日まで）を超えている場合は接種不可。
  </StepFlowStep>
  <StepFlowStep number={3} title="同時接種プランの作成">
    今回: ヒブ3回目 + PCV3回目 + 4種混合3回目 + B肝3回目 + BCG（計5本同時接種）。4週間後: 必要に応じて追加接種。
  </StepFlowStep>
  <StepFlowStep number={4} title="今後のスケジュール">
    1歳到達後: MR1回目 + 水痘1回目 + ヒブ4回目 + PCV4回目の同時接種を計画。
  </StepFlowStep>
</StepFlow>

<PromptTemplate title="キャッチアップスケジュール作成">

- 3ヶ月: ロタ1価2回目、ヒブ2回目、PCV2回目、B肝2回目、4種混合2回目
- 4ヶ月以降: 未接種

以下の条件でスケジュールを組んでください：
1. 可能な限り早く追いつくこと
2. 同時接種を積極的に活用
3. 各ワクチンの最低接種間隔を遵守

</PromptTemplate>
