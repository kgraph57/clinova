コミュニティとリソース — 学び続けるために

公式ドキュメント、GitHub、プラグインマーケットプレイス、コミュニティ。Claude Codeエコシステムの最新情報をキャッチアップする方法を扱う。

技術は日々進化する。Claude Codeも例外ではない。本書で学んだ知識を土台に、最新情報をキャッチアップし続ける方法を紹介する。

公式リソース

ドキュメント

Anthropicの公式ドキュメントが最も信頼できる情報源。

  • Claude Code ドキュメント: インストール、設定、使い方の公式ガイド
  • Claude API リファレンス: APIの詳細仕様
  • Agent SDK ドキュメント: エージェント構築の公式ガイド

GitHub

  • anthropics/claude-code: Claude Code本体のリポジトリ。Issueで問題報告やフィードバックが可能
  • Release Notes: 新機能やバグ修正の履歴

プラグインエコシステム

Everything Claude Code (ECC)

コミュニティで最も利用されているプラグイン。

  • 37以上のスキル: /tdd, /code-review, /security-review 等
  • 13のエージェント定義: planner, architect, tdd-guide 等
  • 31のコマンド: 開発ワークフローの自動化
  • GitHub: affaan-m/everything-claude-code

カスタムプラグインの作成と共有

自分で作ったスキルやエージェントをプラグインとしてパッケージ化し、チームやコミュニティと共有できる。

~/.claude/plugins/marketplaces/my-plugin/
├── skills/
│   ├── my-skill/
│   │   └── SKILL.md
│   └── another-skill/
│       └── SKILL.md
├── agents/
│   └── my-agent.md
└── README.md

コミュニティ

ソーシャルメディア

  • X（Twitter）: #ClaudeCode タグで最新のTipsや活用事例が共有されている
  • YouTube: チュートリアル動画やライブコーディングセッション

フィードバック

Claude Codeの改善に貢献する方法:

  1. GitHub Issues: バグ報告や機能リクエスト（https://github.com/anthropics/claude-code/issues）
  2. プラグイン公開: 自作のスキルやエージェントを公開
  3. 記事・ブログ: 活用事例を共有

学習パス

初学者（本書読了後）

  1. 毎日の開発でClaude Codeを使い、操作に慣れる
  2. CLAUDE.mdを充実させ、プロジェクト固有の最適化を進める
  3. Hooksを1つずつ追加し、ワークフローを自動化する

中級者

  1. カスタムSkillを作成し、繰り返し作業をコマンド化する
  2. カスタムAgentを定義し、コードレビューやセキュリティチェックを自動化する
  3. MCPサーバーを設定し、外部サービスとの連携を構築する

上級者

  1. Agent SDKでカスタムエージェントを構築する
  2. マルチエージェントシステムを設計し、チーム開発を自動化する
  3. CI/CDパイプラインにAIエージェントを統合する
  4. コミュニティにプラグインやスキルを公開する

医療×Claude Codeの未来

医療分野でのAIコーディングは、まだ始まったばかりだ。

可能性

  • 電子カルテ統合: AIが直接カルテシステムと連携し、定型業務を自動化
  • 臨床意思決定支援: リアルタイムの診療支援ツールの構築
  • 研究自動化: 文献検索から統計解析までのパイプライン自動化
  • 医学教育: インタラクティブな症例学習ツールの開発

課題

  • 規制対応: 医療機器としてのAIツールの規制
  • プライバシー: 患者データの安全な取り扱い
  • 信頼性: AIの出力の検証と品質保証
  • 公平性: バイアスのないAIツールの開発

おわりに

本書を通じて、Claude Codeとそのエコシステムの全体像を学んだ。

インストールから始まり、ツールシステム、CLAUDE.md、Hooks、Skills、Agents、MCP、マルチエージェント開発、そしてClaude APIとAgent SDK。

これらは道具にすぎない。重要なのは、あなたがこの道具で何を作るかだ。

医療の現場を少しでもよくするアプリケーション。患者さんの理解を助ける説明資料。研究を加速するツール。チームの生産性を高めるワークフロー。

Claude Codeは、あなたのアイデアを形にする力を持っている。

あとは、手を動かすだけだ。
