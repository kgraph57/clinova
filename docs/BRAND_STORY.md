# Hoshizu BRAND STORY

「BRAND STORYの作り方」フレームワーク（MISSION → VALUE → VISION）に基づき、Hoshizu の存在意義・価値観・目指す世界を整理したドキュメントです。

---

## ① MISSION（ミッション）

**わたしたちは何のために存在しているのか？**  
→ **（過去）企業としての存在意義**

> 医療AIに関する情報は、SNS・論文・書籍・ブログ・学会など、あちこちに「星」のように散らばっている。  
> それらを**つなぎ、実践で使える「星座」に変える**こと。  
> つまり、**散らばる星を、星座にする**ために、Hoshizu は存在する。

- **一言**: 医療現場で使えるAIナレッジを、バラバラの情報から「星座」として体系化し、届ける。
- **問いへの答え**: 「何のために？」→ 医療従事者が「明日の診療・研究・教育でそのまま使える」形でAIナレッジにアクセスできるようにするため。

---

## ③ VALUE（バリュー）

**なぜそれが実現できるのか？**  
→ **（現在）組織として重要視する価値観・原則**

Hoshizu が MISSION を果たし VISION に近づくために、日々の判断とコンテンツづくりの基準にしている価値観。

| 価値観 | 内容 |
|--------|------|
| **実践第一** | 「読んで終わり」にしない。プロンプトテンプレート・Playground・コースで、その場で試し、明日の臨床で使える形にする。 |
| **エビデンスと誠実さ** | 医療コンテンツは根拠を明示し、参考文献を載せる。診断・治療の最終判断は医師に委ねる旨を忘れない。 |
| **3つの交差点** | 臨床医療・AI/テクノロジー・教育・メディアの3領域が交わる場所でしか生まれない「使える知」を大切にする。 |
| **オープンと再利用** | ナレッジはMDXで管理し、ポータル・PWA・RSS・ニュースレターで届ける。できるだけオープンに、再利用しやすく。 |
| **継続的に更新** | 一度作って終わりにしない。ニュース取得・週次ピックアップ・コース更新で、コンテンツを生きている状態に保つ。 |

- **問いへの答え**: 「なぜ実現できる？」→ 上記の価値観でコンテンツとプロダクトを選び、作り、直し続けているから。

---

## ② VISION（ビジョン）

**わたしたちは何を目指しているのか？**  
→ **（未来）企業として実現したい世界**

> すべての医療従事者が、AIを**怖れず、正しく、日常のツール**として使いこなし、  
> その分だけ**患者さんにより良い医療**を届けられる世界。

- **一言**: 医療AIの「星座」が当たり前に参照され、現場の意思決定・文書・研究・教育の一部になっている状態。
- **問いへの答え**: 「何を目指す？」→ 医療従事者が「AI、どこで使うか」ではなく「どう使うか」だけを考えればよい世界。

---

## ストーリーの流れ（図の矢印に対応）

```
① MISSION（なぜ存在するか）
    「散らばる星を、星座にする」— バラバラの医療AI情報を、実践で使える形に体系化して届ける
        ↓
③ VALUE（なぜそれができるか）
    実践第一・エビデンス・3領域の交差点・オープン・継続更新 — 日々の判断とコンテンツの基準
        ↓
② VISION（何を目指すか）
    すべての医療従事者がAIを日常のツールとして使い、患者により良い医療を届けられる世界
```
