# マルチエージェント協調

## このテクニックとは

複数の専門家役割で協調して分析する

マルチエージェント協調は、複数の専門家役割で協調して分析する手法です。

## 基本的な使い方
複数の専門家の視点から分析を促します。

例：
\

## 医療での活用例

### シナリオ

高度な合併症を持つ患者が複数の診療科をまたぐ症例で入院している。内科、外科、放射線科の専門医が連携して診断と治療方針の策定を求められている状況。

### プロンプト例

```text
複数の専門家として役割を分担し、協力して分析してください。
```

## いつ使うべきか

- 極めて複雑な症例で多角的な分析が必要な場合
- より高度な分析や専門的な出力が必要な場合に検討してください。
