# ガイドライン要約

## このテクニックとは

臨床ガイドラインは数百ページに及ぶことも多く、緊急の臨床場面で必要な情報を素早く引き出すのは難しい。AIに「診断基準・重症度分類・治療アルゴリズム・推奨グレード」という構造化された要約を依頼することで、素早くガイドラインの核心を把握できる。

特に有効なのは、学会を横断した比較だ。「日本循環器学会・ACC/AHA・ESCのガイドラインでの推奨の違いを比較して」と指定すると、国際比較の視点で情報が整理される。海外の論文を読む際や、国際学会での発表を準備する際に役立つ。

ただし、AIのガイドライン知識にはカットオフ時期があり、最新の改訂が反映されていない可能性がある。「このガイドラインが改訂された可能性がある場合は記載してほしい」と付け加え、重要な臨床判断の前は必ず原典を確認する習慣が重要だ。

## 基本パターン

```text
[ガイドライン名]（発行年：[年]）について、以下の要点を構造化して要約してください。

1. 診断基準（主要診断基準・副次診断基準・診断に必要な検査）
2. 重症度分類（分類の軸・各重症度の定義）
3. 治療アルゴリズム（重症度別・病期別の治療の流れ）
4. 推奨グレード（Class I/IIa/IIb/III、またはGrade A/B/Cなど）
5. 特に注意すべき点・よくある誤解
6. 前版からの主な改訂点（把握している場合）

このガイドラインが改訂された可能性がある場合は、その旨を記載してください。
```

## 医療での活用例

### シナリオ

ガイドラインの要点を素早く確認したい。

### プロンプト例

```text
[ガイドライン名]について、以下の要点を要約してください：

- 診断基準
- 重症度分類
- 治療アルゴリズム
- 推奨グレード（あれば）
- 特に注意すべき点
```

## いつ使うべきか

- 専門外の疾患を担当し、関連ガイドラインの骨格を素早く把握したいとき
- カンファレンス・症例検討会の発表前にガイドライン準拠の治療方針を確認するとき
- 研修医・医学生へのティーチングでガイドラインの重要ポイントを解説するとき
- 日本と海外のガイドラインの推奨の違いを比較・把握したいとき
- ガイドラインが改訂されたことを聞き、前版との違いを効率よく確認したいとき
