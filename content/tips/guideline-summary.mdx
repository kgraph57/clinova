# ガイドライン要約

## このテクニックとは

ガイドラインの要点を要約させる

長いガイドラインの要点を素早く把握できます。

## 基本パターン

\

## 医療での活用例

### シナリオ

ガイドラインの要点を素早く確認したい。

### プロンプト例

```text
[ガイドライン名]について、以下の要点を要約してください：

- 診断基準
- 重症度分類
- 治療アルゴリズム
- 推奨グレード（あれば）
- 特に注意すべき点
```

## いつ使うべきか

- ガイドラインの確認
- より高度な分析や専門的な出力が必要な場合に検討してください。
