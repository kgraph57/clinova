# クリニカルクエスチョン整理

## このテクニックとは

臨床疑問をPICO形式に整理させる

臨床疑問をPICO形式に整理し、文献検索しやすくします。

## 基本パターン

\

## 医療での活用例

### シナリオ

臨床疑問を文献検索したい。検索式がうまく作れない。

### プロンプト例

```text
以下の臨床疑問をPICO形式に整理してください：

臨床疑問: [疑問]

出力：
- P (Patient)
- I (Intervention)
- C (Comparison)
- O (Outcome)
- PubMed検索式の提案
```

## いつ使うべきか

- 文献検索の準備、抄読会
- より高度な分析や専門的な出力が必要な場合に検討してください。
