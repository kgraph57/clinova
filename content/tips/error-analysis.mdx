# エラー分析依頼

## このテクニックとは

インシデントの要因分析を依頼

インシデントやヒヤリハットの要因分析ができます。

## 基本パターン

\

## 医療での活用例

### シナリオ

インシデントレポートを書く。再発防止策を考えたい。

### プロンプト例

```text
以下のインシデントについて要因分析をしてください：

インシデント内容: [内容]
発生状況: [状況]

分析項目:
- 直接原因
- 背景要因（4M分析）
- 再発防止策
- システム改善の提案
```

## いつ使うべきか

- インシデント分析、医療安全
- より高度な分析や専門的な出力が必要な場合に検討してください。
