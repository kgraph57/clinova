# ロールプレイプロンプト

## このテクニックとは

ロールプレイプロンプトは、AIに特定の役割（ロール）を割り当てることで、その役割にふさわしい専門性、視点、文体で回答を生成させるテクニックです。「あなたは〇〇です」と指定するだけで、AIはその専門家になりきって応答します。

このテクニックが効果的な理由は、AIの膨大な学習データの中から、指定された役割に関連する知識パターンが優先的に活性化されるためです。例えば「総合診療医」と指定すれば、総合診療に関連する臨床知識、診断アプローチ、表現パターンが強調されます。

## 基本的な構造

```text
あなたは[役割]です。[追加の背景情報]

[タスクの指示]
```

役割の指定に加えて、経験年数、専門領域、働く環境などの背景情報を追加すると、より具体的で実用的な回答が得られます。

## 具体例

### シンプルなロール指定

```text
あなたは経験豊富な総合診療医です。
以下の症例について、鑑別診断を提示してください。

症例: [症例情報]
```

### 詳細なロール指定（より効果的）

```text
あなたは大学病院の感染症内科で15年の臨床経験を持つ
感染症専門医です。抗菌薬適正使用支援チーム（AST）の
リーダーとして、院内の抗菌薬使用について指導を行っています。

以下の症例について、抗菌薬の選択と投与設計を提案してください。
エビデンスに基づいた根拠も併せて説明してください。

症例: [症例情報]
```

### 複数のロールを活用する例

```text
以下の症例について、3つの専門科の視点から分析してください。

1. 総合診療医の視点: 全体的な鑑別診断と初期対応
2. 循環器内科医の視点: 心血管イベントのリスク評価
3. 内分泌内科医の視点: 代謝性疾患の可能性

症例: [症例情報]
```

## 医療で特に有効なロール設定

### 専門医コンサルテーション

地域のクリニックで専門医にすぐに相談できない場合、AIに専門医のロールを指定して意見を求めることができます。

```text
あなたは経験豊富な小児神経科専門医です。
地域のかかりつけ医から以下の相談を受けました。

「5歳男児。3ヶ月前から時々ぼんやりすることがあり、
親御さんが心配されています。数秒間動きが止まり、
声をかけても反応しません。その後、何事もなかったように
活動を再開します。頻度は1日に2-3回程度です。」

この所見から考えられる疾患と、かかりつけ医が行うべき
初期評価、専門医への紹介の必要性について助言してください。
```

### 患者説明の練習

```text
あなたは患者教育に長けた糖尿病専門医です。
新たに2型糖尿病と診断された60歳の患者さんに、
以下の内容を医学用語を使わずにわかりやすく説明してください。

1. 2型糖尿病とはどのような病気か
2. なぜ治療が必要か
3. 食事と運動の重要性
4. 薬の役割
```

## メリットとデメリット

### メリット

- **専門的な視点の獲得**: 指定した専門領域に特化した回答が得られる
- **一貫性のある文体**: 同じロールを維持することで、文体やトーンが安定する
- **多角的な分析**: 複数のロールを使い分けることで、異なる視点を得られる
- **簡単に実装可能**: 一文追加するだけで効果がある

### デメリット

- **過度な専門化**: ロール設定が狭すぎると、重要な一般的視点を見落とす場合がある
- **ロールの限界**: AIは実際の臨床経験を持たないため、微妙なニュアンスは再現できない
- **バイアスの可能性**: 特定のロールに偏った回答が生成される場合がある

## 効果的なロール設定のコツ

1. **具体的に指定する**: 「医師」よりも「循環器内科の専門医」の方が効果的です
2. **経験レベルを明記する**: 「10年の臨床経験を持つ」などの補足が回答の質を向上させます
3. **環境を設定する**: 「大学病院」「地域のクリニック」など、働く環境を指定すると実用的な回答が得られます
4. **目的を明確にする**: ロール設定と併せて、何を求めているかを明確に伝えます

## いつ使うべきか

- 専門医に相談できない状況で専門的な意見が欲しい場合
- 患者説明のための平易な表現を準備したい場合
- 症例を多角的に分析したい場合
- 一貫した文体やトーンで文書を作成したい場合
