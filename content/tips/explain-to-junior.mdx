# 「研修医に説明して」法

## このテクニックとは

AIに「研修医1年目に説明するように」と対象読者を指定すると、専門用語を噛み砕いた平易な解説が生成される。難解な論文・複雑な病態生理・最新の治療概念を理解したいとき、このテクニックは即効性がある。

「研修医向け」という指定が効く理由は、AIが「読者が知っていること・知らないこと」の想定を変えるからだ。内科専門医向けなら当然の前提として省略される基礎知識が、研修医向けでは丁寧に補足される。これにより、学習者だけでなく専門家も「改めて基礎から整理する」目的で活用できる。

さらに、この手法は「患者向け説明」の下書きにも転用できる。研修医向けより一段階噛み砕いた「患者さんに説明するように」という指定と組み合わせると、インフォームドコンセントの資料作成にも応用が効く。

## 基本パターン

```text
以下の内容を、研修医1年目でも理解できるように説明してください。

【説明してほしい内容】
[論文の内容・疾患の概念・治療メカニズム・検査の原理など]

説明の際は以下を守ってください：
- 専門用語を使う場合は、初出時に簡単な解説を括弧内に加える
- 理解を助けるアナロジーや身近な例を使う
- 「なぜそうなるのか」の病態生理を含める
- 臨床での実践に直結するポイントを最後にまとめる
```

## 医療での活用例

### シナリオ

複雑な論文や症例を理解したい。患者説明の資料を作りたい。

### プロンプト例

```text
「HFpEF（駆出率が保たれた心不全）の病態生理と治療戦略」について、
研修医1年目でも理解できるように説明してください。

特に以下のポイントを含めてください：
- HFrEFとの違いを病態生理から説明する
- なぜ治療が難しいのか
- SGLT2阻害薬が有効とされるメカニズム
- 診察室でこの診断を疑うべき患者像

専門用語には解説を添え、できるだけ具体的な例を使ってください。
```

## いつ使うべきか

- 難解な論文や総説を読んだあと、内容を自分の言葉で整理・理解したいとき
- 研修医や医学生に複雑な疾患概念をティーチングする資料を準備するとき
- 患者さんへのインフォームドコンセントの説明草案を素早く作りたいとき
- 専門外の領域の疾患を担当し、基礎から学び直す必要があるとき
- 自分がすでに知っている内容の「説明力」を鍛える自己学習のとき
