# 処方チェック依頼

## このテクニックとは

多剤併用（ポリファーマシー）が問題となる高齢患者や、腎・肝機能低下のある患者への処方では、薬物相互作用や用量調整の必要性を見落とすリスクがあります。AIは添付文書・ガイドラインの知識を大量に保持しており、「この処方の組み合わせで問題はないか」という確認作業を高速に行うことができます。

ただしAIの回答には情報の鮮度や hallucination のリスクがあるため、重要な判断はDI（医薬品情報）センターへの問い合わせや添付文書の確認で補完することが必須です。このテクニックはあくまで「見落としに気づくための補助ツール」として活用し、AIの回答をそのまま処方根拠とすることは避けてください。

チェック精度を上げるコツは、患者背景（腎機能・肝機能・年齢・体重・妊娠の有無）を必ず一緒に伝えることです。腎機能の情報があれば、腎排泄性薬剤の用量適切性まで確認できます。

## 基本パターン

```text
以下の処方内容について、薬剤師の視点でチェックしてください。

【患者背景】
- 年齢・性別・体重: [例: 78歳 女性 48kg]
- 腎機能: [例: eGFR 35 mL/min/1.73m²（CKD G3b）]
- 肝機能: [例: 正常範囲内]
- 主な既往・診断: [例: 2型糖尿病、慢性心不全（EF 40%）、高血圧]
- アレルギー: [例: ペニシリン系アレルギー（発疹歴）]

【新規追加予定の処方】
[例: メトホルミン500mg 1回1錠 1日2回（朝夕食後）]

【現在の継続処方】
[例:
- フロセミド20mg 1T/日
- スピロノラクトン25mg 1T/日
- カルベジロール10mg 2T/日
- リシノプリル5mg 1T/日
- アトルバスタチン10mg 1T/日
]

以下の観点でチェックしてください:
1. 薬物相互作用（PK/PD両面）
2. 腎機能・肝機能に基づく用量適切性
3. 禁忌・慎重投与への該当
4. 注意すべき副作用（特にモニタリングが必要なもの）
5. 重複投与・同効薬の確認

問題があれば重要度（要注意 / 注意）で分類して示してください。
```

## 医療での活用例

### シナリオ

複数薬剤を処方する前に、相互作用が気になる。

### プロンプト例

```text
以下の処方内容について確認してください：
- 薬物相互作用
- 用量の妥当性（腎機能考慮）
- 禁忌・慎重投与
- 注意すべき副作用

処方内容: [処方]
患者情報: [年齢、腎機能、併存疾患、併用薬]
```

## いつ使うべきか

- CKDや肝硬変など臓器機能低下のある患者への新規薬剤追加前
- 5剤以上を併用する高齢患者の定期処方見直し（ポリファーマシー対策）
- 他科から処方された薬剤との相互作用を手術・検査前に確認したいとき
- 抗菌薬・抗凝固薬・免疫抑制剤など相互作用リスクの高い薬剤を開始するとき
- 夜間・休日など薬剤師へ相談できない状況での一次スクリーニングとして
