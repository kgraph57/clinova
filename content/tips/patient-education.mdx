# 患者説明文作成

## このテクニックとは

患者説明文作成は、医師が伝えたい医療情報を患者の理解度・年齢・背景に合わせた平易な言葉でAIに変換させる手法です。

専門用語で記述された医療情報を「患者目線」にリライトすることで、インフォームドコンセント資料・疾患説明パンフレット・退院指導文書など、患者と向き合う場面で使える資料を素早く作成できます。

## 基本パターン

```text
以下の情報をもとに、患者さん向けの説明文を作成してください。

【説明したい内容】
[疾患名/治療名/処置名]

【対象患者】
- 年齢層: [例: 70代高齢者 / 就学前の子どもの親 / 働き盛りの成人]
- 理解度: [例: 医療知識なし / 基本的な知識あり]

【含めてほしい内容】
- この病気/治療はどういうものか（1〜2文で端的に）
- なぜこの治療が必要か
- 具体的な治療/処置の流れ
- 注意事項・気をつけること
- よくある質問とその回答

【文体・形式の条件】
- 専門用語は避け、日常語で説明
- 一文を短く（40字以内を目安）
- 箇条書きを活用して読みやすく
- 必要に応じて「〇〇とは…」の解説を先頭に加える
```

## 医療での活用例

### シナリオ①：糖尿病患者へのインスリン自己注射指導

外来で初めてインスリン自己注射を始める70代の患者に、自宅で読み返せる説明文を用意したい。

```text
以下の内容で患者向け説明文を作成してください。

【説明したい内容】
インスリン自己注射の方法と注意点

【対象患者】
- 年齢層: 70代、一人暮らし
- 理解度: 医療知識ほぼなし

【含めてほしい内容】
- インスリンが必要な理由
- 注射のタイミング（食前 vs 就寝前）
- 実際の注射手順（ステップごとに）
- 低血糖の症状と対処法
- Q&A: 針を忘れた/血が出た/量を間違えた

【文体・形式の条件】
- ひらがな多めで平易に
- 各ステップに番号をつける
- 緊急連絡先を最後に追記
```

### シナリオ②：大腸内視鏡前処置の説明（30代患者）

```text
大腸内視鏡検査の前処置について、患者向け説明文を作成してください。

対象: 30代、医療知識は一般社会人レベル

含めてほしい内容:
- 検査2日前からの食事制限（食べてよいもの・いけないもの）
- 前日の流動食メニュー例
- 当日の下剤の飲み方（時間・量・コツ）
- 準備完了の目安（便の色）
- よくある不安: 腹痛・寒気はどこまでOKか
- 検査後の食事再開タイミング

文体: フレンドリーだが丁寧に。Q&A形式を一部取り入れると◎
```

### シナリオ③：既存カルテ記載のリライト

電子カルテのコメントをそのまま貼り付けてリライトさせる最速パターン。

```text
以下の医療記録をもとに、患者さんが読めるわかりやすい説明文に書き直してください。

【原文（カルテ記載）】
Dx: 2型糖尿病 HbA1c 9.2% → 内服強化＋インスリン導入方針
Rx: グラルギン U-100 10単位 就寝前 + メトホルミン 750mg/日継続

【要件】
- 専門用語を平易に
- 「なぜ注射が必要か」を最初に説明
- 300字程度でまとめる
```

## 精度を上げるコツ

### 対象読者を具体的に指定する

「患者向け」だけでなく、**年齢・知識レベル・生活背景**まで指定するとより適切な文体になります。

```text
# 粗い指定（NG）
対象: 患者

# 良い指定
対象: 80代女性、認知機能は保たれているが文字を読む機会が少ない。
      娘（40代）が読んで本人に説明することが多い。
```

### 「含めないでほしいこと」も指定する

AIは情報を網羅しようとするため、不要な情報で長くなりがちです。

```text
# 以下は含めないでください
- 手術リスクの統計数値（不安を煽るため）
- 保険適用の詳細（窓口で別途説明するため）
```

## いつ使うべきか

- **外来診療前**: よく使う疾患説明の雛形を一度作っておく
- **退院指導**: 内服・食事・受診タイミングをまとめて説明
- **インフォームドコンセント補助**: 患者が事前に予習できる資料
- **多言語対応**: 翻訳と説明簡略化を同時に依頼

## 注意事項

- AIが作成した説明文は必ず医師が確認・修正してから使用してください
- 個人情報（患者名・ID）はプロンプトに含めないようにしましょう
- 疾患の統計数値・生存率などは最新ガイドラインと照合が必要です
