# 時間的推論

## このテクニックとは

時間的推論は、患者の状態やデータを「ある時点のスナップショット」としてではなく、「時間の流れの中での変化」として分析させる手法です。同じ検査値でも、「上昇傾向にある」「急速に悪化している」「横ばいで安定している」では臨床的な意味が全く異なります。AIに時系列データを提示し、トレンドの分析と将来予測を求めることで、経時的変化に基づく洗練された臨床判断を支援します。

医療では疾患の「経過」が診断にも治療にも不可欠な情報です。例えば、胸痛の発症から受診までの時間はACS治療方針に直結し、発熱の経過日数は感染症の重症度と抗菌薬の適応に影響します。また、慢性疾患のフォローアップでは「前回比でどう変わったか」「この1年でのトレンドは？」という時間軸での評価が治療方針変更の根拠になります。

時間的推論をAIに行わせる際のコツは、データに「時刻・日付・病日」を明示し、変化の速度（急性・亜急性・慢性）を意識させることです。「この変化は○日間で起きた」「前回から○週間が経過した」という時間情報を含めることで、AIは単なる数値の羅列でなく、臨床的に意味のある変化パターンを認識した分析を提供します。

## 基本的な使い方

```text
以下の経時的データを分析し、時間の流れに沿って変化の意味を解釈してください。

【患者情報】
[基本情報]

【経時的データ】
[日付/病日]: [検査値・バイタル・症状]
[日付/病日]: [検査値・バイタル・症状]
[日付/病日]: [検査値・バイタル・症状]

分析してほしいこと:
1. 各指標のトレンド（改善・悪化・安定・変動）の把握
2. 変化の速度から推測される病態の性質（急性/慢性）
3. 今後3ヶ月・6ヶ月の予測（現在のトレンドが続いた場合）
4. トレンドの変化点（改善から悪化への転換など）があれば、その意味を考察
5. 治療介入が効果を発揮しているかどうかの評価
```

予後予測テンプレート：

```text
以下の慢性疾患患者の経過から、今後のリスクを予測してください。

【患者】
[年齢・性別・診断名・罹病期間]

【過去の経過（時系列）】
[時点1]: [状態・検査値・治療内容]
[時点2]: [状態・検査値・治療内容]
[時点3（現在）]: [状態・検査値・治療内容]

予測してほしいこと:
- 現在のトレンドが続いた場合の1年後・5年後の予測
- リスクが高まっている合併症とそのタイムライン
- 現時点で介入すれば改善できる要因
- 次の外来（[X週後])までにモニタリングすべき指標
```

## 医療での活用例

### シナリオ

慢性疾患を抱える患者のフォローアップ外来で、過去数回の検査データや症状の変化を踏まえて今後の病状進行や合併症リスクを評価する必要がある。医師は時間軸に沿ったデータ解析を基に治療方針の見直しを検討している。

### プロンプト例

```text
以下の[患者情報]と[過去の検査結果や症状の経時的変化データ]を用いて、時間の経過に沿って病状の進行をステップバイステップで解析してください。その上で、今後の予後予測と合併症リスクについて具体的な考察を示し、推奨される治療方針の変更点を提案してください。異常な変化や重要な指標のトレンドがあれば詳細に説明してください。

【患者情報】
58歳男性、2型糖尿病（罹病10年）・高血圧・脂質異常症

【経時的データ】
2年前: HbA1c 7.2%、eGFR 72、尿アルブミン/Cr比 25mg/g、BP 132/82
1年前: HbA1c 7.8%、eGFR 65、尿アルブミン/Cr比 65mg/g、BP 138/86
現在: HbA1c 8.4%、eGFR 58、尿アルブミン/Cr比 145mg/g、BP 142/90

現在の治療: メトホルミン・シタグリプチン・アムロジピン

トレンドから読み取れる病態変化と、今後1年で必要な治療強化の提案をしてください。
```

## いつ使うべきか

- 慢性疾患（糖尿病・CKD・心不全・COPD）の定期フォローアップで経過を評価したいとき
- 入院患者の病態が改善・悪化しているかどうかをデータで客観的に評価したいとき
- 敗血症・急性腎障害など急性期疾患で「悪化速度」から重症度を評価したいとき
- 治療介入の効果を前後比較で定量的に評価したいとき
- 複数の検査値の変化パターンから潜在的な合併症の進行を早期発見したいとき
