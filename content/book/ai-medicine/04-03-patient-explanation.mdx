**医師が話す言葉と、患者が理解できる言葉には、埋めがたい溝がある。AIは、この溝を埋める。**

---

## 患者説明における3つの課題

### 課題1: 専門用語の氾濫

医師が普段使う言葉は、患者には難しすぎる。

**医師の説明**:
```
お子さんは急性上気道炎です。ウイルス性を疑います。
対症療法として鎮咳去痰薬を処方します。
3日後に再診してください。増悪時は速やかに受診してください。
```

**患者（保護者）が理解できない部分**:
- 急性上気道炎 → ？
- ウイルス性 → 抗生物質は効かないの？
- 対症療法 → ？
- 鎮咳去痰薬 → ？
- 増悪 → ？

**実際の理解**: 「風邪で薬をもらった。3日後に来る。」

### 課題2: 読解レベルの不一致

日本人の平均的な読解力は、**小学5-6年生レベル**とされている。

しかし、多くの医療文書は**高校卒業〜大学レベル**で書かれている。

読めない文書は、読まれない。
読まれない文書は、理解されない。
理解されない文書は、意味がない。

### 課題3: 情報量の過多

医師は「すべて説明した」ことで安心する。
患者は「情報が多すぎて覚えられない」と感じる。

**認知心理学の知見**: 人間が一度に記憶できる情報は**3-5項目**まで。

10項目を説明しても、患者が覚えているのは最初と最後の2-3項目だけである。

**AIは、これら3つの課題すべてに対応できる。**

## 患者説明文書作成の5ステップ・プロトコル

---

## Step 1: 説明内容の医学的整理

### 目的
まず、医学的に正確な内容を整理する（Ch.3-4で学んだ技術を使う）。

### プロンプトの型

```
【Role】あなたは小児科専門医です。

【Task】以下の診断・治療について、医学的に正確な説明内容を整理してください。

【Context】
診断: 急性上気道炎（ウイルス性を疑う）
患者: 3歳男児
症状: 3日前から咳嗽、昨夜から発熱38.5℃
検査: 溶連菌(-)、インフルエンザ(-)
治療: カルボシステイン、アセトアミノフェン頓用

【Format】
以下の項目について整理:
1. 病気の正式名称と病態
2. 原因
3. 今後の経過（治癒までの期間）
4. 治療の目的と方法
5. 自宅でのケア
6. 注意すべき症状（再診の目安）

【Constraint】
- この段階では専門用語を使用してOK
- 医学的に正確であることを最優先
```

### AI出力（Step 1）

```
【医学的整理】

1. 病気の正式名称と病態
   - 急性上気道炎（Common Cold）
   - 鼻腔から喉頭までの上気道粘膜の炎症
   - 主にウイルス感染による

2. 原因
   - ウイルス感染（ライノウイルス、コロナウイルス、アデノウイルスなど）
   - 細菌感染は否定的（溶連菌検査陰性）
   - インフルエンザも否定的

3. 今後の経過
   - 通常3-7日で自然治癒
   - 咳は1-2週間持続することあり
   - 発熱は1-3日で解熱

4. 治療の目的と方法
   - 原因療法: なし（ウイルス性のため）
   - 対症療法: 症状を和らげる
     ・カルボシステイン（去痰薬）: 痰を出しやすくする
     ・アセトアミノフェン（解熱鎮痛薬）: 発熱・不快感を和らげる

5. 自宅でのケア
   - 水分摂取
   - 安静
   - 加湿
   - 栄養摂取（食欲に応じて）

6. 注意すべき症状（再診の目安）
   - 発熱が3日以上持続
   - 呼吸困難（息が苦しそう）
   - 水分摂取困難
   - ぐったりして元気がない
   - 咳が2週間以上持続
```

## Step 2: 平易な言葉への変換

### 目的
Step 1の内容を、小学3-4年生でも理解できる言葉に変換する。

### プロンプトの型

```
【Role】あなたは患者教育の専門家です。

【Task】Step 1の医学的内容を、小学3-4年生でも理解できる平易な言葉に変換してください。

【Context】
[Step 1の出力]

読者: 3歳児の保護者（医学知識なし、小学校卒業程度の読解力を想定）

【Format】
以下の項目について、平易な言葉で記載:
1. どんな病気か（2-3文）
2. 原因は何か（1-2文）
3. どのくらいで治るか（1文）
4. 薬は何のために使うか（各1文）
5. 自宅でできること（箇条書き3-5項目）
6. こんな時はすぐ病院へ（箇条書き3-5項目）

【Constraint】
- 専門用語は使わない（使う場合は必ず（ ）で説明）
- 1文は30字以内
- 小学3-4年生が読める漢字のみ使用（ふりがなをつける）
- カタカナ語も最小限に
- 抽象的な表現を避け、具体的に
```

### AI出力（Step 2）

```
【平易な言葉での説明】

1. どんな病気か
   お子さんは「かぜ」です。
   鼻やのどに、ばい菌（ウイルス）がついて、赤くはれています。
   多くの子どもがかかる病気で、心配いりません。

2. 原因は何か
   ウイルスという、とても小さなばい菌です。
   抗生物質（細菌をやっつける薬）は効きません。

3. どのくらいで治るか
   3日から1週間で治ります。

4. 薬は何のために使うか
   ■ たんを出しやすくする薬（カルボシステイン）
   せきをすると、のどから「たん」が出ます。
   この薬は、たんをやわらかくして、出しやすくします。

   ■ 熱を下げる薬（アセトアミノフェン）
   熱が高くてつらそうな時に使います。
   熱が下がると、少し楽になります。

5. 自宅でできること
   □ 水やお茶をたくさん飲ませてください
   □ 部屋をあたたかく、しめらせてください（加湿）
   □ むりに食べさせなくてもいいです（食べたいものを）
   □ たくさん寝かせてあげてください

6. こんな時はすぐ病院へ
   □ 熱が3日以上続く
   □ 息が苦しそう、ゼーゼーする
   □ 水やお茶を飲めない
   □ ぐったりして、元気がない
   □ せきが2週間以上続く
```

## Step 3: 読みやすさの測定と調整

### 目的
文章の読みやすさを客観的に測定し、必要に応じて調整する。

### プロンプトの型

```
【Role】あなたは文章の読みやすさ評価の専門家です。

【Task】以下の文章の読みやすさを評価し、改善案を提示してください。

【Context】
[Step 2の出力]

【Format】
1. 読みやすさの評価指標
   - 平均文長（目標: 30字以内）
   - 使用漢字のレベル（小学3-4年生配当漢字か）
   - カタカナ語の数
   - 専門用語の数

2. 改善が必要な箇所
   - 長すぎる文
   - 難しい漢字
   - わかりにくい表現

3. 改善案

【Constraint】
- 目標: 小学3-4年生が読める
- 医学的な正確性は損なわない
- 簡潔さと理解しやすさのバランス
```

### AI出力（Step 3）

```
【読みやすさ評価】

1. 評価指標
   - 平均文長: 27字 ✓（目標30字以内）
   - 使用漢字: 小学3-4年配当 ✓
   - カタカナ語: 5個（ウイルス、アセトアミノフェン、カルボシステイン）
   - 専門用語: 実質0個 ✓

2. 改善が必要な箇所

   ⚠️ 長い文:
   「この薬は、たんをやわらかくして、出しやすくします。」
   → 2文に分割推奨

   ⚠️ カタカナ語:
   「アセトアミノフェン」「カルボシステイン」
   → 一般名は残し、役割を強調

   ⚠️ わかりにくい表現:
   「加湿」
   → 「しめらせる」に変更済みだが、さらに具体的に

3. 改善案

   【改善前】
   この薬は、たんをやわらかくして、出しやすくします。

   【改善後】
   この薬を飲むと、たんがやわらかくなります。
   せきをした時に、たんが出やすくなります。

   ---

   【改善前】
   部屋をあたたかく、しめらせてください（加湿）

   【改善後】
   部屋をあたたかくしてください。
   ぬれたタオルをほすか、加湿器を使うと、のどが楽になります。

   ---

   【改善前】
   熱を下げる薬（アセトアミノフェン）

   【改善後】
   熱を下げる薬（カロナール、アンヒバなど）
   ※商品名の方が親しみやすい
```

## Step 4: 視覚的要素の追加

### 目的
文字だけでなく、視覚的要素（図、箇条書き、色分け）を加えて理解を助ける。

### プロンプトの型

```
【Role】あなたは医療情報デザインの専門家です。

【Task】Step 3の文章に、視覚的要素を追加して理解しやすくしてください。

【Context】
[Step 3の改善後の文章]

【Format】
以下の要素を追加:
1. セクションごとの見出し（アイコン付き）
2. 重要ポイントの強調（太字、色）
3. チェックリスト形式
4. 症状の重症度別の色分け（緑: 様子見、黄: 注意、赤: 緊急）

【Constraint】
- A4用紙1枚に収まる
- 印刷しても見やすい
- カラーが使えない場合も考慮（白黒でも識別可能）
```

## Step 5: 理解度の確認

### 目的
説明文書が本当に理解されるか、事前にテストする。

### プロンプトの型

```
【Role】あなたは患者教育の評価専門家です。

【Task】以下の説明文書について、理解度確認のための質問を作成してください。

【Context】
[Step 4の最終版]

【Format】
以下の3種類の質問を各3問作成:

1. 知識の確認（○×問題）
   例: 「この病気は抗生物質で治る」○か×か？

2. 理解の確認（選択問題）
   例: 熱が何℃以上の時に解熱薬を使いますか？
   a) 37℃  b) 38℃  c) 38.5℃  d) 39℃

3. 応用の確認（状況判断）
   例: 以下のうち、すぐ病院に行くべきなのはどれ？

【Constraint】
- 説明文書を読めば答えられる問題
- 難しすぎない（小学3-4年生レベル）
```

### AI出力（Step 5）

```
【理解度確認の質問】

## 1. 知識の確認（○×問題）

Q1: この病気は抗生物質で治る。○か×か？
正解: × （ウイルス性なので抗生物質は効かない）

Q2: たんを出す薬を飲むと、せきが止まる。○か×か？
正解: × （たんを出しやすくする薬で、せき止めではない）

Q3: 熱が3日以上続いたら、病院に行く。○か×か？
正解: ○

## 2. 理解の確認（選択問題）

Q1: 熱が何℃以上の時に解熱薬を使いますか？
a) 37℃  b) 38℃  c) 38.5℃  d) 39℃
正解: c) 38.5℃

Q2: この病気は何日くらいで治りますか？
a) 1日  b) 3-7日  c) 2週間  d) 1ヶ月
正解: b) 3-7日

Q3: 部屋の温度は何℃がいいですか？
a) 15℃  b) 20-25℃  c) 30℃  d) 書いていない
正解: b) 20-25℃

## 3. 応用の確認（状況判断）

Q1: 以下のうち、すぐ病院に行くべきなのはどれ？
a) 熱が1日続いている
b) せきが3日続いている
c) 息がゼーゼーして苦しそう
d) 食欲が少し落ちている

正解: c) 息がゼーゼーして苦しそう

Q2: 自宅でできることは、次のうちどれ？
a) 水をたくさん飲ませる
b) むりに食べさせる
c) 部屋を寒くする
d) 寝かせない

正解: a) 水をたくさん飲ませる

Q3: 薬について正しいのはどれ？
a) 熱がなくても解熱薬を飲む
b) たんの薬は1日3回飲む
c) 抗生物質をもらう
d) 薬を飲めば1日で治る

正解: b) たんの薬は1日3回飲む
```

### 実例2: 川崎病（5歳児）— より複雑な病気

**Step 1-4を省略し、Step 5（最終版）のみ提示**

```markdown
# お子さんの病気について（川崎病）

## どんな病気？

お子さんは「**川崎病**」です。

全身の血管に炎症（はれ）が起きる病気です。

日本では年間15,000人くらいの子どもがかかります。

**きちんと治療すれば、ほとんどの子は元気になります。**

---

## 原因は？

まだよくわかっていません。

ばい菌やウイルスではないので、うつりません。

---

## いつ治る？

治療を始めれば、**2-3日で熱が下がります**。

ただし、心臓の検査を続ける必要があります。

---

## 治療について

### 点滴の薬（免疫グロブリン）
**何のため**: 炎症を止めて、心臓を守るため

体の中の「炎症をやっつける力」を強くする薬です。

1回だけ、12時間かけて点滴します。

### 飲み薬（アスピリン）
**何のため**: 血が固まるのを防ぐため

最初は1日3回、熱が下がったら1日1回に減らします。

何ヶ月か飲み続けます。

---

## 入院について

**入院が必要です**（5-7日くらい）

- 点滴の治療をします
- 心臓の検査（エコー）をします
- 毎日、熱や症状をチェックします

---

## 心臓について（一番大事）

川崎病で心配なのは、**心臓の血管（冠動脈）が太くなること**です。

でも、きちんと治療すれば、**95%以上の子は心臓に問題が残りません**。

### 心臓の検査スケジュール
- 入院中: 1回
- 退院時: 1回
- 退院後: 2週間後、1ヶ月後、3ヶ月後、6ヶ月後、1年後

**必ず検査に来てください。とても大事です。**

---

## 退院後の生活

退院したら、ふつうの生活ができます。

- 保育園・幼稚園: 行ってOK
- 外遊び: してOK
- お風呂: 入ってOK
- 予防接種: 半年後から（医師に相談）

---

## こんな時はすぐ病院へ

| 症状 | 緊急度 |
|------|--------|
| 胸が痛いと言う | **今すぐ救急車** |
| 息が苦しそう | **今すぐ救急車** |
| 顔色が悪い、ぐったり | **今すぐ** |
| また高い熱が出た | **すぐ連絡** |

---

## 連絡先

- 病棟直通: 03-XXXX-XXXX（内線1234）
- 外来: 03-XXXX-YYYY
- 夜間・休日: 03-XXXX-ZZZZ（救急外来）
```

**ポイント**:
- 病気の重大性を伝えつつ、不安を煽らない
- 「95%以上は問題なし」と具体的数値で安心させる
- 「心臓の検査が大事」と最重要点を強調
- フォローアップスケジュールを明示

### 実例3: 予防接種（6ヶ月児）— 保護者の不安に対応

```markdown
# 予防接種について（BCG）

## 今日受ける予防接種

**BCG**（結核を防ぐ注射）

---

## どんな病気を防ぐ？

**結核（けっかく）**という病気を防ぎます。

結核は、せきで広がる病気です。

赤ちゃんがかかると、重症になることがあります。

---

## 注射の方法

ふつうの注射と違います。

**スタンプのように、9本の針を左うでに押します。**

「ハンコ注射」とも言います。

少しチクッとしますが、すぐ終わります。

---

## 注射の後、どうなる？

### 1-2週間後
- うでに赤いポツポツができます → **正常な反応です**
- かゆがることがあります → かかないように

### 3-4週間後
- ポツポツが少しもり上がります → **正常です**
- じくじくすることがあります → **正常です**

### 3ヶ月後
- かさぶたができて、はがれます
- 小さなあとが残ります（一生残ります）

**これは、ワクチンが効いている証拠です。心配いりません。**

## 自宅でのケア

### してもいいこと
- お風呂に入る（注射の日から入れます）
- せっけんで洗う
- ふつうに抱っこする

### してはいけないこと
- 注射のあとをこする
- 注射のあとに薬をぬる（医師の指示がない限り）
- バンソウコウを貼る

A: 4週間後に、四種混合を受けられます。
```

## まとめ

- **5ステップ・プロトコル** — 医学的整理 → 平易化 → 読みやすさ測定 → 視覚化 → 理解度確認
