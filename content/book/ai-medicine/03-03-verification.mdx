## 検証の3層構造 — Surface / Evidence / Logic

AIの出力を検証するには、3つの層がある。

### Layer 1: Surface Check（表層確認）— 5秒でできる

最も基本的なチェック。AIの出力を受け取った瞬間に行う。

**チェック項目**:
1. 数値が含まれているか → 含まれていれば要注意
2. 固有名詞（論文名、ガイドライン名）があるか → あれば要検証
3. 「〜と考えられています」など曖昧な表現があるか → あれば出典を求める
4. 出力が長すぎないか → 長いほどハルシネーションが紛れ込みやすい

**この層で引っかかったら、Layer 2へ進む。**

### Layer 2: Evidence Check（エビデンス確認）— 5分でできる

出力に含まれる「事実」を、独立した情報源で確認する。

**チェック方法**:
1. 数値 → 添付文書、ガイドライン、教科書で確認
2. 論文引用 → PubMed、Google Scholarで実在確認
3. ガイドライン → 学会公式サイトで最新版確認
4. 統計データ → 元データソース（厚労省、WHO等）で確認

**この層でエビデンスが見つからなければ、Layer 3へ進む。**

### Layer 3: Logic Check（論理確認）— 10分でできる

出力の「論理構造」を批判的に吟味する。

**チェック項目**:
1. 因果関係は妥当か（相関を因果と混同していないか）
2. 前提条件は明示されているか
3. 例外や限界は述べられているか
4. 代替解釈の可能性はないか

**この層で論理的矛盾が見つかれば、出力を棄却するか、プロンプトを修正して再質問。**

## 1つの疑問を3つの方法で検証する

検証の基本原則は、**単一情報源に依存しない**ことである。

以下の実例で、3層構造の検証を実践してみよう。

---

## 【実例1】小児の解熱薬投与量

### AI出力

```
質問: 3歳児（体重15kg）へのアセトアミノフェンの投与量は？

AI回答:
アセトアミノフェンの小児投与量は、体重1kgあたり10-15mg、
1回量として投与します。3歳児15kgの場合、1回150-225mgが適切です。
投与間隔は4-6時間ごと、1日最大投与量は体重1kgあたり60mgです。
```

### Layer 1: Surface Check（5秒）

- 数値が含まれている（10-15mg/kg、60mg/kg/日）
- 具体的すぎる数値（150-225mg）
- → **Layer 2へ進む**

### Layer 2: Evidence Check（5分）

**検証方法1: 添付文書で確認**
- カロナール錠・細粒の添付文書を確認
- 記載: 「通常、アセトアミノフェンとして、1回10-15mg/kg、1日総量として60mg/kgを限度」
- → **1回量と1日総量の記載は正しい**

**検証方法2: 小児薬用量一覧で確認**
- 日本小児科学会「小児薬用量一覧」
- 記載: 同上
- → **独立した情報源でも一致**

**検証方法3: 複数のAIに聞く**
- ChatGPT、Claude、Geminiに同じ質問
- 全て同じ用量を回答
- → **複数AIで一致**

### Layer 3: Logic Check（10分）

**論理的妥当性**:
- 1回10-15mg/kg、最大4回/日 → 40-60mg/kg/日となり、上限60mg/kgと整合
- 投与間隔4-6時間 → 薬理学的半減期（2-3時間）から妥当
- → **論理的に一貫**

### 検証結果: 正解

この出力は信頼できる。3層すべてで矛盾がない。

## 【実例2】川崎病の診断基準（Ch.2-3の復習）

### AI出力

```
質問: 川崎病の診断基準を教えてください。

AI回答:
川崎病の診断基準は、5日以上の発熱に加えて、
以下の6つの主要症状のうち5つを満たすことです：
(1)両側眼球結膜充血、(2)口唇・口腔所見、(3)発疹、
(4)四肢末端の変化、(5)頸部リンパ節腫脹、(6)BCG接種部位の発赤。
```

### Layer 1: Surface Check（5秒）

- 数値が含まれている（6つ、5つ）
- 具体的な症状名
- → **Layer 2へ進む**

### Layer 2: Evidence Check（5分）

**検証方法1: 日本小児科学会ガイドライン**
- 「川崎病診療ガイドライン2020年改訂版」を確認
- 記載: 「5つの主要症状のうち4つ以上」
- → **「6つのうち5つ」は誤り**

**検証方法2: 複数のAIに聞く**
- ChatGPT: 「5つのうち4つ」と回答
- Claude: 「5つのうち4つ」と回答
- → **他のAIと不一致、元のAIがハルシネーション**

**検証方法3: BCG発赤の位置づけ**
- ガイドライン確認: BCG発赤は「参考所見」であり、主要症状ではない
- → **BCGを主要症状に含めている時点で誤り**

### Layer 3: Logic Check（10分）

- 「6つのうち5つ」では、診断感度が下がる（見逃しリスク増加）
- 実臨床での運用と矛盾
- → **論理的に不合理**

### 検証結果: ハルシネーション

この出力は誤り。Layer 2で明確に検出された。

## 【実例3】新生児低血糖の治療閾値（Ch.2-3の復習）

### AI出力

```
質問: 新生児の低血糖の治療開始基準は？

AI回答:
新生児低血糖の治療閾値は血糖値50mg/dL未満とされています。
無症候性の場合でも、この値を下回れば治療を開始します。
```

### Layer 1: Surface Check（5秒）

- 数値（50mg/dL）
- 「〜とされています」という曖昧な表現
- → **Layer 2へ進む**

### Layer 2: Evidence Check（5分）

**検証方法1: AAP（米国小児科学会）ガイドライン**
- "Postnatal Glucose Homeostasis in Late-Preterm and Term Infants" を確認
- 記載: 治療閾値は時間と症状で変動
  - 生後0-4時間: 25mg/dL未満
  - 生後4-24時間: 35mg/dL未満
  - 生後24時間以降: 45mg/dL未満（無症候性の場合）
- → **「一律50mg/dL」は誤り**

**検証方法2: 日本の基準**
- 日本小児科学会「新生児低血糖症の管理」
- 記載: 同様の時間経過別基準
- → **日米で基準が一致、AIの回答は簡略化しすぎ**

**検証方法3: 複数のAIに時間を指定して聞く**
- 「生後2時間の新生児の低血糖治療閾値は？」と具体化
- Claude: 「25mg/dL未満」と正確に回答
- → **元の質問が曖昧だったため、AIが簡略化した可能性**

### Layer 3: Logic Check（10分）

- 新生児は生理的に低血糖傾向（胎盤からのグルコース供給が途絶える）
- 時間経過とともに血糖調節機構が成熟
- → **時間別の基準がある方が生理学的に妥当**

### 検証結果: 不正確（ハルシネーションではないが、簡略化しすぎ）

この出力は「間違い」ではないが、「不十分」である。プロンプトを改善すべき。

**改善版プロンプト**:
```
【Role】あなたは新生児科医です。

【Task】正期産新生児の低血糖治療閾値を、生後時間別に説明してください。

【Context】
- 対象: 正期産新生児（37週以上）
- 状況: 無症候性低血糖
- 日本とAAP両方のガイドラインを参照

【Format】
表形式で、以下を記載:
| 生後時間 | 治療閾値（mg/dL） | 根拠ガイドライン |

【Constraint】
- 「〜とされています」ではなく、具体的なガイドライン名を明記
- 有症候性と無症候性を区別
```

## 【実例4】統計的有意差の解釈

### AI出力

```
質問: この論文では p&lt;0.05 で有意差ありとされていますが、
臨床的に意味がありますか？

論文の結果: 新薬群で収縮期血圧が平均2mmHg低下、p=0.03

AI回答:
p=0.03であり、統計的に有意な差が認められます。
したがって、この新薬は臨床的に有効であると言えます。
```

### Layer 1: Surface Check（5秒）

- p値の解釈
- 「臨床的に有効」という結論
- → **統計と臨床の混同の可能性、Layer 2へ**

### Layer 2: Evidence Check（5分）

**検証方法1: 効果量の確認**
- 収縮期血圧2mmHg低下の臨床的意義は？
- 一般的に、収縮期血圧10mmHg低下で心血管イベント20%減少とされる
- 2mmHg低下では、イベント減少は約4%程度
- → **統計的有意だが、臨床的には小さい効果**

**検証方法2: 信頼区間の確認**
- 論文の95%CI を確認（仮に: -3.5 to -0.5 mmHg）
- 上限が-0.5mmHgということは、効果がほぼゼロの可能性もある
- → **統計的有意でも、効果の大きさには不確実性**

**検証方法3: NNT（治療必要数）の計算**
- 4%のイベント減少 → NNT = 100/4 = 25
- つまり、25人治療して1人のイベント予防
- コストや副作用を考慮すると、費用対効果は？
- → **臨床的意義は限定的**

### Layer 3: Logic Check（10分）

- AIは「p&lt;0.05 = 臨床的に有効」という論理的誤謬を犯している
- 統計的有意差と臨床的意義は別物
- → **論理的誤り**

### 検証結果: 論理的誤り

この出力は、統計的有意差を臨床的意義と混同している。

**正しい回答例**:
```
p=0.03で統計的に有意ですが、効果量（2mmHg）は臨床的には小さいです。
95%CI、NNT、費用対効果を総合的に評価する必要があります。
統計的有意差のみで臨床的有効性は判断できません。
```

## 【実例5】ガイドラインの最新性

### AI出力

```
質問: 小児肺炎の治療ガイドラインは？

AI回答:
日本小児呼吸器学会「小児呼吸器感染症診療ガイドライン2017」に基づくと、
市中肺炎の第一選択薬はアモキシシリンです。
```

### Layer 1: Surface Check（5秒）

- 固有名詞（ガイドライン名）
- 年号（2017）
- → **最新性の確認が必要、Layer 2へ**

### Layer 2: Evidence Check（5分）

**検証方法1: 学会公式サイトで最新版確認**
- 日本小児呼吸器学会サイトを確認
- 最新版: 「小児呼吸器感染症診療ガイドライン2022」が公開されている
- → **2017年版は古い、2022年版がある**

**検証方法2: 2022年版での変更点確認**
- 2022年版を確認
- アモキシシリンは依然として第一選択
- ただし、耐性菌の動向、COVID-19後の変化などが追加記載
- → **結論は同じだが、文脈が更新されている**

**検証方法3: AIに「2025年現在の最新版は？」と再質問**
- Claude: 「2022年版が最新です」と回答
- → **AIの訓練データが古い場合がある**

### Layer 3: Logic Check（10分）

- 2017年から2025年まで8年経過
- この間に薬剤耐性、パンデミック、新薬登場など変化あり
- → **ガイドラインが更新されている可能性は高い**

### 検証結果: 情報が古い

結論（第一選択薬）は正しいが、最新版を参照すべき。

## 検証の時間配分 — 10分ルール

「検証に時間がかかりすぎる」という声がある。

しかし、検証は**必ずしも完璧である必要はない**。

### 重要度別の検証時間

| 重要度 | 例 | 検証時間 | 検証レベル |
|--------|-----|----------|-----------|
| **Critical** | 投薬量、診断基準、緊急対応 | 10分 | 3層すべて |
| **High** | 治療方針、ガイドライン引用 | 5分 | Layer 1-2 |
| **Medium** | 一般的知識、背景情報 | 2分 | Layer 1のみ |
| **Low** | 患者説明の言い回し、雑学 | 0分 | 検証不要 |

**10分ルール**: Critical項目でも、10分以内に検証を終える。それ以上かかるなら、別の方法（専門医コンサルト、文献精読）を検討。

**Ch.2-3で「疑い深く」なった。Ch.3-2で「聞き方」を学んだ。Ch.3-3で「確かめ方」を学んだ。**
