## 批判的吟味の3つの誤解

まず、多くの医師が持つ誤解を解いておこう。

### 誤解1: 「批判的吟味 = 論文のアラ探し」

**違う。**

批判的吟味とは、論文の結論を鵜呑みにせず、以下を問うことである:

- この研究デザインは妥当か？
- バイアスのリスクはどの程度か？
- 結果は私の患者に適用できるか？
- 代替的な解釈はないか？

「批判」は「否定」ではなく、「慎重な評価」である。

### 誤解2: 「批判的吟味には統計学の専門知識が必要」

**部分的に正しいが、完全ではない。**

統計学の知識は有用だが、最も重要なのは**「構造化された質問」**である。

AIは統計計算を補助できる。しかし、「この研究に臨床的意義があるか」という判断は、医師にしかできない。

### 誤解3: 「批判的吟味は時間がかかる」

**従来は正しかった。AIの時代では違う。**

従来、1本の論文を批判的に読むには60-90分かかった。AIを使えば、構造化された吟味が**15-20分**で完了する。

## 批判的吟味の6ステップ・フレームワーク

以下のフレームワークは、JAMA Users' Guides to the Medical LiteratureとCochrane Handbookを基に、AI活用に最適化したものである。

---

## Step 1: 研究の妥当性（Internal Validity）

### 目的
この研究は、内部的に妥当か？つまり、結果は信頼できるか？

### プロンプトの型

```
【Role】あなたは臨床疫学の専門家です。

【Task】以下の論文について、研究の妥当性（Internal Validity）を評価してください。

【Context】
[論文のPDF全文、またはURL]

【Format】
以下の項目について評価:

1. 研究デザインの適切性
   - リサーチクエスチョンに対して適切なデザインか？
   - RCT/コホート/症例対照のどれが最適か？
   - 実際のデザインは最適か？

2. ランダム化（RCTの場合）
   - ランダム化の方法は記載されているか？
   - Concealment（割付の隠蔽）は適切か？
   - ベースラインの患者特性は両群で同等か？

3. 盲検化
   - 患者の盲検化はあるか？
   - 評価者の盲検化はあるか？
   - 解析者の盲検化はあるか？

4. フォローアップの完全性
   - 脱落率は何%か？
   - 脱落の理由は記載されているか？
   - ITT（Intention-to-treat）解析が行われているか？

5. 交絡因子の調整
   - 主要な交絡因子は特定されているか？
   - 多変量解析で調整されているか？

6. 総合評価: バイアスのリスク（高/中/低）

【Constraint】
- Cochrane Risk of Bias toolに準拠
- 「不明」も選択肢（論文に記載がない場合）
- 数値は正確に抽出
```

### 実例: 小児の鉄欠乏性貧血RCT

**論文**: Gupta A et al. (2023) "Iron dosing in pediatric anemia: A randomized trial"

**AI出力（Step 1）**:

```
【研究の妥当性評価】

1. 研究デザインの適切性: ✓ 適切
   - リサーチクエスチョン: 小児IDAに対する至適投与量
   - 最適デザイン: RCT
   - 実際のデザイン: 二重盲検RCT → 適切

2. ランダム化: ✓ 低リスク
   - 方法: コンピュータ生成の乱数表（記載あり）
   - Concealment: 封筒法、薬剤師が管理（適切）
   - ベースライン特性:
     年齢: 6mg群 18.3±6.2ヶ月 vs 3mg群 17.8±5.9ヶ月 (p=0.65)
     Hb: 6mg群 8.9±0.8g/dL vs 3mg群 9.1±0.7g/dL (p=0.18)
     → 両群で同等

3. 盲検化: ✓ 低リスク
   - 患者: 盲検化あり（製剤の外観を同一にした）
   - 評価者: 盲検化あり（Hb測定は独立した検査技師）
   - 解析者: 盲検化あり（統計解析は第三者機関）

4. フォローアップの完全性: ⚠️ 中リスク
   - 脱落率: 6mg群 25%（15/60名）、3mg群 12%（7/60名）
   - 脱落理由: 副作用による中断（6mg群10名、3mg群3名）、
             転居（6mg群3名、3mg群2名）、フォローアップ拒否（6mg群2名、3mg群2名）
   - ITT解析: なし（Per-protocol解析のみ）
   → ⚠️ 脱落率が高く、ITT解析がない（質を下げる要因）

5. 交絡因子の調整: ✓ 低リスク
   - 主要交絡因子: 年齢、性別、ベースラインHb、栄養状態
   - 多変量解析: あり（調整済み）

6. 総合評価: バイアスのリスク = 中
   理由: 脱落率が高い（特に6mg群）、ITT解析なし
   影響: 効果を過大評価している可能性あり（副作用で中断した患者が解析から除外）
```

**ポイント**: AIは構造的な評価を提供するが、「臨床的にどう解釈するか」は医師が判断する。

## Step 2: 結果の大きさと精度（Magnitude &amp; Precision）

### 目的
統計的有意差だけでなく、効果の大きさと信頼区間を評価する。

### プロンプトの型

```
【Role】あなたは生物統計学の専門家です。

【Task】以下の論文の結果について、効果の大きさと精度を評価してください。

【Context】
[論文のResults section]

【Format】
以下を抽出・計算:

1. 主要アウトカムの結果
   - 介入群の結果（平均±SD、または割合）
   - 対照群の結果（平均±SD、または割合）
   - 群間差（95%信頼区間）
   - p値

2. 効果の大きさ
   - 連続変数の場合: 平均差、Cohen's d
   - 二値変数の場合: リスク比（RR）、オッズ比（OR）、リスク差（RD）

3. 臨床的意義
   - NNT（治療必要数）またはNNH（有害必要数）
   - MCID（最小臨床的重要差）と比較

4. 信頼区間の解釈
   - 95%CIの幅は狭いか広いか？
   - 信頼区間にMCIDが含まれるか？

5. 統計的有意差 ≠ 臨床的意義
   - p&lt;0.05だが臨床的には意味が小さい場合を指摘

【Constraint】
- 数値は論文から正確に抽出
- 計算ミスがないか検算
- 「統計的有意だから臨床的に重要」という誤謬を避ける
```

### AI出力（Step 2）

```
【効果の大きさと精度】

1. 主要アウトカム: Hb改善量（12週後）
   - 6mg群: +2.8±0.9 g/dL (n=45)
   - 3mg群: +2.3±0.8 g/dL (n=53)
   - 群間差: 0.5 g/dL (95%CI: 0.1-0.9)
   - p値: 0.04 → 統計的有意

2. 効果の大きさ
   - 平均差: 0.5 g/dL
   - Cohen's d: 0.58（中程度の効果量）

3. 臨床的意義
   - MCID（最小臨床的重要差）: 小児IDAでは一般的に1.0 g/dL
   - 観察された差: 0.5 g/dL → **MCIDの半分**
   - NNT: 計算不能（連続変数のため、二値化が必要）
   - 仮にHb≧11g/dLを「治癒」と定義した場合:
     6mg群: 73%（33/45）が治癒
     3mg群: 64%（34/53）が治癒
     絶対リスク差: 9%
     NNT: 11（11人治療して1人追加で治癒）

4. 信頼区間の解釈
   - 95%CI: 0.1-0.9 g/dL
   - CIの幅: 0.8 g/dL（やや広い）
   - 下限0.1 g/dLは臨床的にほぼ意味なし
   - 上限0.9 g/dLでもMCID（1.0 g/dL）に届かない
   → **真の効果がMCIDに達しない可能性が高い**

5. 統計的有意差 ≠ 臨床的意義
   ⚠️ 重要な指摘:
   - p=0.04で統計的に有意
   - しかし効果の大きさ（0.5 g/dL）は臨床的に小さい
   - 95%CIの上限でもMCIDに届かない
   - **統計的有意だが、臨床的意義は限定的**
```

**ポイント**: Ch.5-1で学んだ「Clinical Bottom Line」は、このStep 2の評価が基盤になる。

## Step 3: 外的妥当性（External Validity / Generalizability）

### 目的
この研究の結果は、私の患者に適用できるか？

### プロンプトの型

```
【Role】あなたは臨床疫学の専門家です。

【Task】以下の論文の外的妥当性を評価してください。

【Context】
[論文のMethods - 対象患者の記載]

私の患者:
- 3歳男児
- Hb 8.5 g/dL
- 日本在住
- 特記すべき既往歴なし

【Format】
1. 研究対象患者の特徴
   - 年齢範囲、重症度、併存疾患、地域

2. 私の患者との類似点・相違点
   - 類似点:
   - 相違点:

3. 除外基準の確認
   - 私の患者は除外基準に該当するか？

4. 実施地域・医療環境の違い
   - 研究実施国:
   - 医療システムの違い:
   - 利用可能な治療の違い:

5. 外的妥当性の総合評価
   - この結果は私の患者に適用可能か？（高/中/低）
   - 適用する際の注意点

【Constraint】
- 「完全に一致」はあり得ない
- 「どの程度の違いなら許容できるか」を判断
- 患者の価値観・希望も考慮
```

### AI出力（Step 3）

```
【外的妥当性評価】

1. 研究対象患者の特徴
   - 年齢: 6-36ヶ月（中央値18ヶ月）
   - Hb: 7.0-10.0 g/dL（平均9.0 g/dL）
   - 重症度: 軽度〜中等度の貧血
   - 地域: インド（都市部の総合病院）
   - 除外基準: 重度栄養不良、慢性疾患、他の貧血原因

2. 私の患者（3歳男児、Hb 8.5 g/dL、日本）との比較

   類似点:
   - 年齢: 3歳（36ヶ月）→ 研究対象範囲の上限に該当
   - Hb: 8.5 g/dL → 研究対象範囲内（7.0-10.0 g/dL）
   - 鉄欠乏性貧血という診断
   - 既往歴なし

   相違点:
   - 地域: 日本 vs インド
   - 栄養状態: 日本の3歳児は一般的に良好、インドでは栄養不良が多い
   - 医療アクセス: 日本は皆保険、インドは自費診療が主
   - 鉄剤の種類: 日本では硫酸第一鉄・クエン酸第一鉄が主流

3. 除外基準の確認
   - 重度栄養不良: なし ✓
   - 慢性疾患: なし ✓
   - 他の貧血原因: なし ✓
   → 除外基準に該当せず、研究対象として適格

4. 実施地域・医療環境の違い
   - 研究実施国: インド
   - インドの小児: 栄養不良が多く、寄生虫感染も一因
   - 日本の小児: 栄養状態良好、偏食による鉄不足が主因
   → **背景要因が異なる可能性**

5. 外的妥当性: 中〜高

   適用可能性: 高
   理由:
   - 年齢・Hbが研究対象範囲内
   - 鉄欠乏性貧血という診断が同一
   - 経口鉄剤という介入は日本でも実施可能

   注意点:
   - インドの患者は栄養不良が多く、鉄吸収が日本と異なる可能性
   - しかし、鉄剤の薬理作用は地域によらず同一
   - 副作用（消化器症状）の頻度は日本でも同程度と予想
   → **この研究結果は、私の患者に適用可能**
```

## Step 4: 代替的な解釈（Alternative Explanations）

### 目的
著者の結論以外に、この結果を説明できる解釈はないか？

### プロンプトの型

```
【Role】あなたは批判的思考に長けた臨床疫学者です。

【Task】以下の論文の結果について、著者の結論以外の代替的な解釈を提示してください。

【Context】
[論文のResultsとDiscussion]

【Format】
1. 著者の結論
   - 著者は何を結論づけているか？

2. 代替的な解釈（3つ以上）
   各解釈について:
   - 解釈の内容
   - この解釈を支持する根拠
   - この解釈に反する根拠
   - もっともらしさ（高/中/低）

3. バイアスによる代替説明
   - 選択バイアス
   - 情報バイアス
   - 交絡

4. 偶然による可能性
   - p値が0.04と小さいが、多重比較の調整は？
   - 事前に設定したα=0.05に対してギリギリ

5. 最も妥当な解釈
   - 著者の結論と代替解釈のどれが最も妥当か？

【Constraint】
- 「批判」は「否定」ではない
- データに基づいた論理的な代替解釈を提示
- 陰謀論的な解釈は避ける
```

### AI出力（Step 4）

```
【代替的な解釈】

1. 著者の結論
   「6mg/kg/日は3mg/kg/日よりもHb改善効果が大きい」

2. 代替的な解釈

   解釈A: 効果の差は脱落バイアスによる見かけ上のもの

   内容: 6mg群は副作用で25%が脱落し、ITT解析がない。
         脱落した患者は効果が低かった可能性が高い。
         残った75%は「副作用に耐えられた患者」であり、
         効果が高く見えるのは選択バイアス。

   支持する根拠:
   - 6mg群の脱落率25% vs 3mg群12%（2倍）
   - ITT解析なし
   - 脱落理由の10/15が副作用

   反する根拠:
   - 著者はPer-protocol解析で感度分析を実施
   - 脱落患者を「最悪シナリオ」（全員Hb改善なし）と仮定しても
     群間差は0.3 g/dL（p=0.08）で傾向は維持

   もっともらしさ: 中

   ---

   解釈B: 真の効果はあるが、臨床的意義は小さい

   内容: 統計的有意差はあるが、効果の大きさ（0.5 g/dL）は
         MCID（1.0 g/dL）の半分であり、臨床的には重要ではない。

   支持する根拠:
   - 群間差0.5 g/dL &lt; MCID 1.0 g/dL
   - 95%CIの上限0.9 g/dLでもMCIDに届かない
   - NNT=11は大きくない

   反する根拠:
   - 「臨床的意義」は患者・状況により異なる
   - 重症貧血ではわずかな改善も重要

   もっともらしさ: 高

   ---

   解釈C: 効果はあるが、副作用とのバランスで推奨されない

   内容: 6mg/kgは効果が大きいが、副作用が2倍、脱落率も2倍。
         「効果」だけでなく「害」も考慮すべき。

   支持する根拠:
   - 副作用: 45% vs 22%（NNH=4）
   - 脱落率: 25% vs 12%
   - 効果の差はわずか0.5 g/dL

   反する根拠:
   - 副作用は一時的で重篤ではない

   もっともらしさ: 高

3. バイアスによる代替説明

   選択バイアス: ✓ あり
   - Per-protocol解析により、脱落患者が除外
   - 6mg群で効果が過大評価されている可能性

   情報バイアス: 低
   - Hb測定は盲検化された検査技師が実施
   - 測定誤差のリスクは低い

   交絡: 低
   - ランダム化により主要交絡因子は調整済み

4. 偶然による可能性

   - p=0.04は統計的有意（α=0.05）
   - しかしギリギリ
   - 多重比較の調整なし（副次アウトカムが5つある）
   → Bonferroni補正を適用すればα=0.05/6=0.008
   → p=0.04は有意でなくなる
   ⚠️ **偶然による可能性を完全には否定できない**

5. 最も妥当な解釈

   著者の結論「6mg/kg/日は3mg/kg/日より効果が大きい」は、
   **統計的には支持されるが、臨床的意義は限定的**。

   理由:
   - 効果の差（0.5 g/dL）は統計的有意だが、臨床的には小さい
   - 副作用が2倍、脱落率も2倍
   - NNT=11、NNH=4 → 「害」の方が頻度が高い

   推奨:
   **3mg/kg/日から開始し、効果不十分な場合のみ6mg/kg/日へ増量**

   これはCh.5-1のClinical Bottom Lineと一致する。
```

## Step 5: 資金源と利益相反（Funding &amp; COI）

### 目的
研究の資金源や利益相反が、結果に影響を与えていないか確認する。

### プロンプトの型

```
【Role】あなたは研究倫理の専門家です。

【Task】以下の論文の資金源と利益相反を評価してください。

【Context】
[論文のAcknowledgments, Funding, COI statement]

【Format】
1. 資金源
   - 政府/公的機関/企業/その他
   - 具体的な組織名

2. 利益相反の開示
   - 著者の利益相反は開示されているか？
   - 具体的な内容

3. バイアスのリスク
   - 企業資金の場合、結果が企業に有利か？
   - 利益相反がある著者の役割は？

4. 総合評価
   - 資金源・COIによるバイアスのリスク（高/中/低）

【Constraint】
- 「企業資金 = 悪」ではない
- 透明性の高さを評価
- 結果との整合性を確認
```

### AI出力（Step 5）

```
【資金源と利益相反】

1. 資金源
   - Indian Council of Medical Research（政府機関）
   - Grant No: ICMR/2021/12345
   - 企業資金: なし

2. 利益相反の開示
   - 「All authors declare no conflicts of interest」
   - 具体的な開示なし

3. バイアスのリスク
   - 政府資金のみ → 企業バイアスのリスクは低い
   - 利益相反の開示あり（「なし」と明記）
   → 透明性は確保されている

4. 総合評価: 低
   - 資金源は公的機関
   - 利益相反なし
   - バイアスのリスクは低い
```

## Step 6: 総合的な批判的吟味のまとめ

### プロンプトの型

```
【Role】あなたはEBMの専門家です。
```
