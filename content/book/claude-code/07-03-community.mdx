## プラグインエコシステム

### Everything Claude Code (ECC)

コミュニティで最も利用されているプラグイン。

- **37以上のスキル**: /tdd, /code-review, /security-review 等
- **13のエージェント定義**: planner, architect, tdd-guide 等
- **31のコマンド**: 開発ワークフローの自動化
- **GitHub**: affaan-m/everything-claude-code

### カスタムプラグインの作成と共有

自分で作ったスキルやエージェントをプラグインとしてパッケージ化し、チームやコミュニティと共有できる。

```
~/.claude/plugins/marketplaces/my-plugin/
├── skills/
│   ├── my-skill/
│   │   └── SKILL.md
│   └── another-skill/
│       └── SKILL.md
├── agents/
│   └── my-agent.md
└── README.md
```

## 学習パス

### 初学者（本書読了後）

1. 毎日の開発でClaude Codeを使い、操作に慣れる
2. CLAUDE.mdを充実させ、プロジェクト固有の最適化を進める
3. Hooksを1つずつ追加し、ワークフローを自動化する

### 中級者

1. カスタムSkillを作成し、繰り返し作業をコマンド化する
2. カスタムAgentを定義し、コードレビューやセキュリティチェックを自動化する
3. MCPサーバーを設定し、外部サービスとの連携を構築する

### 上級者

1. Agent SDKでカスタムエージェントを構築する
2. マルチエージェントシステムを設計し、チーム開発を自動化する
3. CI/CDパイプラインにAIエージェントを統合する
4. コミュニティにプラグインやスキルを公開する

## 医療×Claude Codeの未来

医療分野でのAIコーディングは、まだ始まったばかりだ。

### 可能性

- **電子カルテ統合**: AIが直接カルテシステムと連携し、定型業務を自動化
- **臨床意思決定支援**: リアルタイムの診療支援ツールの構築
- **研究自動化**: 文献検索から統計解析までのパイプライン自動化
- **医学教育**: インタラクティブな症例学習ツールの開発

### 課題

- **規制対応**: 医療機器としてのAIツールの規制
- **プライバシー**: 患者データの安全な取り扱い
- **信頼性**: AIの出力の検証と品質保証
- **公平性**: バイアスのないAIツールの開発
