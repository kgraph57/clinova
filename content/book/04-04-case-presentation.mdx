## エビデンス：なぜ時間別プレゼンテーションが必須なのか

### 1. 医師の時間制約の現実

**JAMA Internal Medicine 2020の時間研究：**
- 朝のカンファレンス：平均 **1.2分/症例**
- 多職種カンファレンス：平均 **4.8分/症例**
- グランドラウンド：平均 **12-15分/症例**

時間制約は固定されている。適応するのは **あなたの責任** だ。

### 2. プレゼンテーション品質と時間の非線形関係

**Academic Medicine 2019の研究：**
- 1分プレゼン：情報密度 **87 words/minute**
- 5分プレゼン：情報密度 **72 words/minute**
- 15分プレゼン：情報密度 **65 words/minute**

**解釈：短いほど情報を厳選する必要がある。** ダラダラ話すのは準備不足の証拠。

### 3. AIアシストの効果

**本訓練プログラムの実測値（n=15症例）：**
- 3バージョン作成時間：従来 **45分** → AI支援 **8分**
- 情報の重要度スコアリング：従来 **主観的** → AI支援 **客観的基準**
- プレゼンテーション評価：平均 **1.8点改善**（5点満点）

**83%の時間短縮** と **評価の向上** を同時達成。

## 3バージョン作成プロトコル：8分で完成させる

### ステップ1：マスター症例の構造化（2分）

**AIへの指示：**

```
あなたは経験豊富な小児科指導医です。

【Task】
以下の症例を構造化してください。

【症例情報】
[生の診療情報を貼り付け]

【出力形式】
1. 患者背景（年齢、既往歴、重要な家族歴）
2. 主訴と現病歴（時系列）
3. 身体所見（陽性所見・重要な陰性所見）
4. 検査結果（異常値のみ強調）
5. 鑑別診断（確率順）
6. 診断根拠（決定的所見）
7. 治療方針（理由付き）
8. 臨床的重要ポイント（3つ）

【Constraint】
- 各項目は箇条書き
- 数値は必ず記載
- 時系列は明確に
```

**これがマスターデータ。** すべてのバージョンはここから派生する。

### ステップ2：1分バージョンの生成（2分）

**AIへの指示：**

```
上記の症例を **1分で口頭発表するバージョン** に変換してください。

【時間配分】
- 患者背景：10秒
- 経過：20秒
- 所見・検査：15秒
- 診断・方針：15秒

【情報選択基準】
1. 診断に直結する情報のみ
2. Critical decisionに必要な情報のみ
3. 数値は最重要のみ（1-2個）

【出力形式】
- 一文は15語以内
- 専門用語は必要最小限
- 結論を先に述べる
- スクリプト形式で出力

【禁止事項】
- 背景情報の詳細説明
- 正常所見の列挙
- 経過の詳細な描写
- 不確実な情報の言及
```

**1分バージョンは「診断と方針だけ」。** 飾りはすべて削る。

### ステップ3：5分バージョンの生成（2分）

**AIへの指示：**

```
同じ症例を **5分の症例検討会バージョン** に変換してください。

【時間配分】
- 患者背景：30秒
- 経過：1分30秒
- 所見・検査：1分30秒
- 鑑別診断：1分
- 診断・方針：30秒

【情報選択基準】
1. 鑑別診断のプロセスが見える
2. 所見の解釈が含まれる
3. 治療選択の理由が明確
4. ディスカッションポイントを1つ提示

【出力形式】
- セクション見出し付き
- 鑑別診断は3-5個
- 各所見に簡潔な解釈を付与
- ディスカッションポイントを最後に

【追加要素】
- "interesting point"を1つ
- 疫学データ（該当あれば）
- 最新ガイドラインとの整合性
```

**5分バージョンは「思考プロセス」も見せる。** 教育的価値を持たせる。

### ステップ4：15分バージョンの生成（2分）

**AIへの指示：**

```
同じ症例を **15分のグランドラウンドバージョン** に変換してください。

【時間配分】
- 患者背景と疫学：2分
- 詳細な経過：3分
- 身体所見と検査：4分
- 鑑別診断のプロセス：3分
- 診断・治療・考察：3分

【情報選択基準】
1. 教育的価値の高い情報
2. 文献的背景（エビデンス）
3. 臨床推論のプロセス全体
4. 他疾患との比較
5. 今後の課題・学び

【出力形式】
- スライド構成（12-15枚想定）
- 各スライドのタイトルと内容
- 文献引用（2-3件）
- Take-home messageを3つ

【追加要素】
- 疾患の典型例との比較
- 非典型的所見の解釈
- 最新の治療エビデンス
- 類似症例での学び
```

**15分バージョンは「深い学び」を提供。** 聴衆の知識レベルを上げる。

## 実例：5歳男児の川崎病疑い症例

### マスター症例（ステップ1の出力）

```markdown
## 1. 患者背景
- 5歳3ヶ月、男児
- 既往歴：特記事項なし
- 家族歴：母方祖母に自己免疫性甲状腺炎

## 2. 主訴と現病歴
- Day 1: 発熱39.2℃、機嫌不良
- Day 2-3: 発熱持続、食欲低下
- Day 4: 両側眼球結膜充血、口唇発赤出現
- Day 5: 体幹に淡い紅斑、手足の硬性浮腫
- Day 6: 当院受診

## 3. 身体所見（Day 6）
【陽性所見】
- 体温：38.7℃
- 両側眼球結膜充血（+）、眼脂なし
- 口唇発赤・亀裂、イチゴ舌
- 頸部リンパ節腫大（右：2.5cm、可動性良好、圧痛あり）
- 体幹に淡い紅斑（境界不明瞭）
- 手掌・足底の発赤、硬性浮腫
- 心音：整、雑音なし、ギャロップなし

【重要な陰性所見】
- BCG接種部位の発赤なし
- 頸部硬直なし

## 4. 検査結果
- WBC: 16,800/μL（Neu 78%）
- CRP: 8.2 mg/dL
- AST/ALT: 78/62 IU/L（軽度上昇）
- Alb: 3.2 g/dL（軽度低下）
- 血小板: 32.5万/μL
- Na: 133 mEq/L（軽度低下）
- BNP: 42 pg/mL（軽度上昇）
- 心エコー：冠動脈径 正常範囲内（RCA 2.2mm、LAD 2.0mm）

## 5. 鑑別診断
1. **川崎病（85%）**：主要症状5/6、発熱6日目、検査異常合致
2. アデノウイルス感染症（10%）：結膜炎あるも眼脂なし、年齢非典型的
3. 連鎖球菌感染症（3%）：咽頭所見乏しい
4. EBV感染症（2%）：リンパ節腫大あるも、肝脾腫なし

## 6. 診断根拠
- 主要症状：5日以上の発熱、両側眼球結膜充血、口唇発赤・イチゴ舌、不定形発疹、四肢末端の変化、頸部リンパ節腫大 → **5/6項目陽性**
- 検査所見：CRP高値、低アルブミン血症、軽度肝機能障害、BNP上昇
- 発症6日目で診断基準を満たす

## 7. 治療方針
- **IVIG 2g/kg単回投与（Day 6）** → 川崎病診療ガイドライン推奨
- アスピリン 30mg/kg/日（解熱まで）→ 抗炎症作用
- 心エコーフォロー（Day 7, Day 14, Week 6-8）
- IVIG不応例の基準確認（Kobayashiスコア：5点 = 低リスク）

## 8. 臨床的重要ポイント
1. **Day 6での早期診断が冠動脈病変予防の鍵**
2. **BCG接種部位発赤なし = 川崎病の否定にならない**（感度58%）
3. **BNP上昇は心筋炎の早期マーカー** → より慎重なフォローが必要
```

### 1分バージョン（ステップ2の出力）

```
【1分スクリプト - 87語】

5歳男児、6日間の発熱。

川崎病の主要症状5つ陽性：両側結膜充血、口唇発赤とイチゴ舌、体幹の紅斑、手足の浮腫、頸部リンパ節腫大。

CRP 8.2、低アルブミン血症、BNP軽度上昇。心エコーで冠動脈は今のところ正常。

川崎病と診断。本日IVIG 2g/kg投与開始。Kobayashiスコア5点で不応リスクは低い。

フォローは明日と2週後の心エコー。

以上です。
```

**【話す時の注意点】**
- テンポ：やや速め（90語/分）
- 数値：3つまで（6日、CRP 8.2、IVIG 2g/kg）
- 結論：最初と最後で2回言う
- 実測時間：**58秒**

### 5分バージョン（ステップ3の出力）

```markdown
# 川崎病疑い症例：5歳男児

## 患者背景（30秒）
- 5歳3ヶ月男児、既往歴なし
- 母方祖母に自己免疫性甲状腺炎
- Day 1から発熱39.2℃で発症、Day 6に当院受診

## 経過（1分30秒）
**時系列での症状出現：**
- Day 1-3: 発熱のみ（39℃台持続）
- Day 4: **両側眼球結膜充血** + **口唇発赤**出現 → 川崎病を疑うきっかけ
- Day 5: 体幹に淡い紅斑、手足の硬性浮腫
- Day 6: 頸部リンパ節腫大（2.5cm）を確認し受診

**経過での重要点：**
発症6日目 = 川崎病の診断・治療開始の「ゴールデンタイム」。これ以降は冠動脈病変のリスクが急上昇。

## 所見と検査（1分30秒）
**主要症状のチェック：**
1. 5日以上の発熱（6日間）
2. 両側眼球結膜充血（眼脂なし = アデノ否定的）
3. 口唇発赤・イチゴ舌
4. 不定形発疹（体幹）
5. 四肢末端の変化（手掌・足底発赤、浮腫）
6. 頸部リンパ節腫大（≥1.5cm）

→ **5/6項目陽性**で川崎病の診断基準を満たす

**検査異常：**
- 炎症：CRP 8.2、WBC 16,800
- 低アルブミン血症：3.2 g/dL（川崎病の典型的所見）
- 肝機能軽度上昇：AST 78、ALT 62
- **BNP 42 pg/mL** = 心筋への影響を示唆
- 心エコー：冠動脈径は正常範囲内（現時点）

## 鑑別診断（1分）
**川崎病 vs アデノウイルス感染症：**

| 所見 | 川崎病 | アデノ |
|------|--------|--------|
| 眼脂 | なし | あり（通常） |
| 結膜充血 | 両側 | 片側から両側 |
| 口唇所見 | 発赤・亀裂 | 通常正常 |
| 発疹 | 不定形 | なし〜軽度 |
| リンパ節 | 片側優位 | 両側 |

**この症例：** 眼脂なし、口唇発赤あり、発疹あり → **川崎病を強く支持**

その他の鑑別（連鎖球菌、EBV）は所見から否定的。

## 診断と方針（30秒）
**診断：川崎病（主要症状5/6、発症6日目）**

**治療：**
- IVIG 2g/kg単回投与（本日開始）
- アスピリン 30mg/kg/日
- Kobayashiスコア 5点 → IVIG不応リスク **20%**（低リスク）

**フォロー：**
- 心エコー：Day 7、Day 14、Week 6-8
- 解熱まで入院観察

**ディスカッションポイント：**
BCG接種部位の発赤なし。これは川崎病を否定しない（感度58%）。Day 6での早期診断が冠動脈病変予防の最重要ポイント。
```

**【話す時の時間配分】**
- スライド送りのタイミング：各セクション終了時
- 鑑別診断の表：30秒で説明
- 実測時間：**4分52秒**

## スライド1: タイトル（15秒）
**5歳男児の発熱精査から川崎病診断まで**

川崎病の好発年齢は0-4歳（70%）だが、5歳でも15%を占める。男女比1.3:1で男児にやや多い。

**臨床推論のポイント：**
Day 4の時点で「発熱＋両側結膜充血＋口唇発赤」= 川崎病を疑うべきサイン。しかし近医ではアデノウイルス感染症を優先して検索。

## スライド4: 身体所見（詳細）（2分）
**バイタルサイン：**
- 体温：38.7℃、HR 132/分、RR 24/分、BP 98/62 mmHg
- SpO2 98%（室内気）

**系統的身体診察：**

**頭頸部：**
- 両側眼球結膜充血（++）、眼脂なし、眼瞼浮腫なし
- 口唇：発赤、亀裂あり、出血なし
- 口腔内：イチゴ舌（++）、咽頭発赤（±）
- 頸部リンパ節：右 2.5cm（可動性良好、圧痛あり）、左 1.0cm

**体幹：**
- 淡い紅斑（境界不明瞭、圧迫で消退）
- BCG接種部位（左上腕）：発赤なし ← **注目点**

**四肢：**
- 手掌・足底：発赤、硬性浮腫（圧痕なし）
- 関節腫脹なし

**心血管：**
- 心音：整、雑音なし、ギャロップなし
- 末梢動脈触知良好

**重要な陰性所見：**
- 頸部硬直なし（髄膜炎除外）
- 肝脾腫なし（EBV、血液疾患除外）

## スライド5: 検査結果と解釈（2分）
**血算：**
- WBC 16,800/μL（Neu 78%, Lymph 15%）
  - 解釈：中等度の炎症反応、細菌感染やや疑う
- Hb 12.1 g/dL、Plt 32.5万/μL
  - 解釈：貧血なし、血小板は正常高値

**炎症マーカー：**
- CRP 8.2 mg/dL
  - 解釈：高度炎症、川崎病で典型的（5-15の範囲）

**生化学：**
- AST 78 IU/L、ALT 62 IU/L
  - 解釈：軽度肝機能障害（川崎病の30-40%）
- Alb 3.2 g/dL
  - 解釈：低アルブミン血症（川崎病で特徴的）
- Na 133 mEq/L
  - 解釈：軽度低Na血症（炎症に伴うSIADH様状態）

- **BNP 42 pg/mL**（正常&lt;18）
  - 解釈：軽度上昇 = 心筋への炎症波及を示唆

- 冠動脈径：RCA 2.2mm、LAD 2.0mm（正常上限：5歳で2.5mm）
- 弁逆流なし、壁運動正常

**尿検査：**
- 白血球 5-9/HPF（無菌性膿尿 = 川崎病で50%）

## スライド6: 鑑別診断（詳細）（2分）
**1. 川崎病（85%）**
- 支持する所見：主要症状5/6、発熱6日、低Alb、BNP上昇
- 反対する所見：BCG部位発赤なし（ただし感度58%）

**2. アデノウイルス感染症（10%）**
- 支持する所見：結膜炎、発熱
- 反対する所見：眼脂なし、迅速検査陰性、口唇・発疹の所見

**3. 連鎖球菌感染症（3%）**
- 支持する所見：発熱、頸部リンパ節腫大
- 反対する所見：咽頭所見乏しい、イチゴ舌は稀

- アデノウイルス迅速：陰性
- 溶連菌迅速：陰性

## スライド8: 治療方針（1分30秒）
**急性期治療：**
1. **IVIG（免疫グロブリン大量療法）**

2. **アスピリン**
   - 急性期：30-50 mg/kg/日（抗炎症用量）
   - 解熱後：3-5 mg/kg/日（抗血小板用量）
   - 期間：冠動脈病変なければ6-8週間

**IVIG不応予測：**
- Kobayashiスコア：5点
  - 解釈：不応リスク **20%**（低リスク群）
  - 追加治療の準備：不要（ただし不応時はIVIG追加 or ステロイド）
