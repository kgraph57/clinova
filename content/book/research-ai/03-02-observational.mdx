# 観察研究の評価

## 観察研究の重要性と限界

医学文献の大部分は観察研究です。RCTが実施できない状況（まれな有害事象、長期予後、倫理的制約）では、観察研究がエビデンスの主要な源泉になります。しかし、観察研究にはRCTにはないバイアスリスクが存在します。

### 観察研究の主な種類

| 研究デザイン | 特徴 | 強み | 弱み |
|-------------|------|------|------|
| 前向きコホート | 曝露群と非曝露群を前向きに追跡 | 時間的順序が明確、発症率を計算可能 | 時間・コストがかかる、脱落バイアス |
| 後ろ向きコホート | 既存データで曝露と結果を評価 | 迅速・低コスト | データの質に依存、記録バイアス |
| ケースコントロール | 症例と対照を比較し曝露を遡及 | まれな疾患に効率的 | リコールバイアス、オッズ比のみ |
| 横断研究 | 一時点での曝露と結果を同時評価 | 迅速、有病率を計算可能 | 因果の方向性不明、一時点のスナップショット |

<Callout>
観察研究の批判的吟味で最も重要なのは「交絡（confounding）」の評価です。RCTではランダム化により交絡因子のバランスが保たれますが、観察研究では意図的に交絡を制御する必要があります。
</Callout>

## STROBEチェックリストによる評価

STROBE（Strengthening the Reporting of Observational Studies in Epidemiology）は、観察研究の報告品質を評価するための22項目のチェックリストです。

[論文テキストを貼り付け]

以下の項目について「報告あり」「部分的」「報告なし」で評価してください：

**タイトル・抄録**

**背景**

**方法**
4. 研究デザインの詳細
5. セッティング（施設、期間、対象者）

17. 一般化可能性
</PromptTemplate>

## Newcastle-Ottawa Scale (NOS) によるバイアス評価

Newcastle-Ottawa Scale は、系統的レビューにおいて非ランダム化研究の質を評価するための最も広く使われたツールです。コホート研究用とケースコントロール研究用があり、それぞれ9点満点で評価します。

### コホート研究のNOS

| カテゴリ | 項目 | 最大スター数 |
|---------|------|------------|
| **Selection** | 曝露群の代表性 | 1 |
| | 非曝露群の選択 | 1 |
| | 曝露の確認方法 | 1 |
| | 研究開始時のアウトカム不在の証明 | 1 |
| **Comparability** | 最も重要な因子での調整 | 1 |
| | その他の因子での調整 | 1 |
| **Outcome** | アウトカムの評価方法 | 1 |
| | 追跡期間の十分さ | 1 |
| | 追跡の完全性 | 1 |

研究デザイン: [コホート / ケースコントロール]
[論文テキストを貼り付け]

**Selection（4項目、各0-1スター）**
1. [コホート] 曝露群の代表性 / [CC] ケースの適切な定義
2. [コホート] 非曝露群の選択 / [CC] コントロールの選択
3. [コホート] 曝露の確認方法 / [CC] コントロールの定義
4. [コホート] 研究開始時のアウトカム不在 / [CC] 曝露とアウトカムの関係

**Comparability（1項目、最大2スター）**
5. 最も重要な交絡因子での調整
6. その他の交絡因子での調整

**Outcome/Exposure（3項目、各0-1スター）**
7. アウトカム/曝露の評価方法
8. 追跡期間 / 症例と対照の応答率
9. 追跡の完全性 / 非応答率

各項目の判定根拠を論文中の記述から引用し、最終スコア（/9）を提示してください。7以上は高品質、4-6は中程度、3以下は低品質と判断します。
</PromptTemplate>

## 交絡の評価

観察研究で最も重要な評価項目は交絡です。

### 交絡の制御方法

<StepFlow>
  <StepFlowStep number={1} title="デザイン段階の制御">
    マッチング、制限（特定の集団に限定）、層別化のいずれかが実施されているか確認。
  </StepFlowStep>
  <StepFlowStep number={2} title="解析段階の制御">
    多変量回帰分析、傾向スコアマッチング、傾向スコア逆確率重み付け（IPTW）、操作変数法等が使われているか確認。
  </StepFlowStep>
  <StepFlowStep number={3} title="調整された交絡因子のリスト">
    どの変数が調整されたかを確認。重要な交絡因子が含まれているか評価。DAG（有向非巡回グラフ）での検討が理想。
  </StepFlowStep>
  <StepFlowStep number={4} title="残余交絡の可能性">
    測定されていない交絡因子の影響を評価。感度分析（E-value等）の有無を確認。
  </StepFlowStep>
</StepFlow>

<PromptTemplate title="交絡の包括的評価">
以下の観察研究における交絡の制御について評価してください。

研究テーマ: [曝露とアウトカムの関係]
[Methodsセクションのテキストを貼り付け]

以下を評価してください：
1. **調整された交絡因子のリスト**: 何が調整されたか
2. **重要な未調整交絡因子**: この研究テーマで調整すべきだが調整されていない因子
3. **交絡の制御方法の妥当性**: 使用された統計手法は適切か
4. **残余交絡の影響**: 未調整交絡がどの方向にバイアスを生むか（過大評価/過小評価）
5. **感度分析の有無**: E-value や その他の感度分析の報告
6. **因果推論の妥当性**: この研究デザインで因果関係を主張できるか
</PromptTemplate>

## 傾向スコア分析の評価

近年の観察研究では、傾向スコア（propensity score）を用いた交絡の調整が増えています。傾向スコア分析の質を評価するポイントを示します。

### 傾向スコア分析のチェックポイント

1. **傾向スコアモデルの変数選択**: アウトカムに関連する変数が含まれているか
2. **バランスの確認**: 傾向スコアマッチング/重み付け後の群間バランス（標準化差&lt;0.1が目安）
3. **分析方法**: マッチング、層別化、IPTW、共変量調整のどれか
4. **重複（overlap）の確認**: 両群の傾向スコア分布に十分な重複があるか
5. **感度分析**: 異なる傾向スコアモデルでの結果の安定性

<Warning>
傾向スコア分析は「万能ではない」ことに注意してください。測定されていない交絡因子は傾向スコアでは制御できません。また、傾向スコアモデルの変数選択が不適切だと、むしろバイアスが増大する可能性があります。
</Warning>

## 観察研究 vs RCT：エビデンスの統合

<ComparisonCard beforeLabel="RCTのみを採用" afterLabel="観察研究も含めて統合">
<ComparisonBefore>

内的妥当性が高い。バイアスリスクが低い。エビデンス数が限られる。一般化可能性に限界。まれな有害事象を検出困難。

</ComparisonBefore>
<ComparisonAfter>

より多くのエビデンスを活用。多様な対象集団をカバー。長期アウトカムのデータを含む。まれな有害事象も検出可能。ただしバイアスリスクの評価が重要。

</ComparisonAfter>

<KeyTakeaway>
観察研究の批判的吟味では、交絡の評価が核心です。STROBEで報告の質を確認し、Newcastle-Ottawa Scaleでバイアスリスクを定量化します。AIはこれらの評価プロセスを効率化しますが、交絡因子の臨床的妥当性の判断や未測定交絡の影響評価は研究者の専門知識が不可欠です。
</KeyTakeaway>
