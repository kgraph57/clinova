# Presentation Outline Creator

## 概要

プレゼンテーションの構成案を作成します。

## プロンプトテンプレート

```text
以下のテーマについて、[duration]分のプレゼンテーションの構成案を作成してください。

# テーマ
[theme]

# 対象聴衆
[audience]

# 伝えたいメッセージ
[key_message]

# 出力形式
【タイトル】
【オープニング（1分）】
【本論】
- セクション1（○分）：
- セクション2（○分）：
- セクション3（○分）：
【クロージング（1分）】
【各スライドの内容案】
```

## 活用のポイント

- 入力情報はできるだけ詳細に記載すると、より精度の高い出力が得られます。
- AIの出力を鵜呑みにせず、必ず専門的知見と照らし合わせて検証してください。
- 必要に応じて出力結果をもとに追加の対話を行い、内容を洗練させてください。
