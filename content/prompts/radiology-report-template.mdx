# 画像所見レポート作成支援

## 概要

放射線科レポートは臨床医への重要な情報伝達手段です。構造化レポートは情報の漏れを防ぎ、臨床医が必要な情報に素早くアクセスできるようにします。

## プロンプトテンプレート

```text
あなたは経験豊富な放射線科専門医です。以下の画像検査について、構造化された読影レポートを作成してください。

# 検査情報
- 検査種別: {{CT/MRI/X線/超音波}}
- 撮影部位: {{部位}}
- 造影: {{単純/造影（造影剤:___）}}
- 臨床情報: {{依頼医からの臨床情報}}
- 比較対象: {{前回検査日（あれば）}}

# 画像所見（箇条書きで入力）
{{観察した所見をそのまま入力}}

# 出力形式
## 検査概要
検査名、撮影条件の要約

## 所見（Findings）
臓器別に構造化して記述：
- 対象臓器/関心領域の所見
- サイズの定量記載（mm単位）
- 前回比較がある場合は変化を記載
- 偶発所見（incidental findings）

## 診断（Impression）
1. 主要診断（臨床的疑問への回答）
2. 鑑別診断（ある場合）
3. 偶発所見への推奨対応
4. フォローアップの推奨（時期と検査方法）

## 推奨
追加検査や臨床的アクションの推奨

注意：AIが生成するレポートは下書きです。最終レポートは放射線科医が確認してください。
```
