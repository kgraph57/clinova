# Medical Certificate Generator

## 概要

診断書・証明書のドラフトを作成します。

## プロンプトテンプレート

```text
以下の内容で[certificate_type]を作成してください。

# 患者情報
[patient_info]

# 診断名
[diagnosis]

# 証明内容
[certificate_content]

# 出力形式
【診断書/証明書】
上記の者は、[diagnosis]にて、[certificate_content]。
期間：[period]

注意：正式な診断書は医師が内容を確認し、署名・押印が必要です。
```

## 活用のポイント

- 入力情報はできるだけ詳細に記載すると、より精度の高い出力が得られます。
- AIの出力を鵜呑みにせず、必ず専門的知見と照らし合わせて検証してください。
- 必要に応じて出力結果をもとに追加の対話を行い、内容を洗練させてください。
