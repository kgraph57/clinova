# 患者説明資料ジェネレーター

## 概要

医療現場では、専門的な内容を患者やご家族にわかりやすく伝える場面が日常的に発生します。しかし、医療者が「わかりやすく説明したつもり」でも、患者側には伝わっていないことが少なくありません。厚生労働省の調査では、処方された薬の飲み方を正しく理解している患者は約6割にとどまるというデータもあります。

このプロンプトは、医学的なトピックを指定すると、専門用語を避けた平易な言葉で患者向けの説明資料を自動生成します。対象者の年齢や理解度に合わせた表現を調整でき、不安を与えすぎず、かつ重要なポイントを確実に伝える資料作りを支援します。

## プロンプトテンプレート

```text
以下の医学的トピックについて、患者やご家族が理解しやすい説明資料を作成してください。

# トピック
[病名・治療法・検査など、説明したい内容]

# 対象者
- 年齢層: [例: 80代の高齢者]
- 説明相手: [患者本人 / 家族 / 両方]
- 理解度の目安: [医療の知識がほとんどない / ある程度の知識がある]

# 作成条件
- 専門用語を使わず、日常の言葉で説明する
- 不安を煽る表現は避けつつ、重要な注意点は明確に伝える
- 箇条書きや表を活用し、視覚的に読みやすくする
- 「こんな時はすぐ相談してください」という受診目安を含める

# 出力形式
1. 病気（または治療・検査）のやさしい説明
2. 日常生活での注意点（具体的な行動レベルで）
3. よくある質問と回答（3つ）
4. こんな時はすぐ相談してください（受診の目安）
```

## 使用例

以下は実際の入力例です。

```text
# トピック
慢性腎臓病（CKD ステージ3b）の食事制限（塩分・カリウム制限）

# 対象者
- 年齢層: 80歳の高齢女性
- 説明相手: 患者本人と同居するご家族（娘さん）
- 理解度の目安: 医療の知識がほとんどない
```

AIは以下のような説明資料を提示します。

### 腎臓の働きについて

> 腎臓は、体の中のいらないものをおしっこと一緒に外に出してくれる臓器です。今、腎臓の元気さが本来の半分くらいになっています。腎臓に負担をかけないように、毎日の食事を少し工夫することが大切です。

### 食事での注意点

**お塩を控えめにしましょう**

- 1日の塩分は小さじ1杯（6g）までが目安です
- お味噌汁は1日1杯までにしましょう
- お漬物やお醤油は、できるだけ控えめにしましょう
- 代わりに、お酢やレモン、しょうが、ごまなどで味つけを工夫してみてください

**カリウムの多い食べ物に気をつけましょう**

- バナナ、メロン、ほうれん草にはカリウムが多く含まれています
- 野菜は茹でこぼす（茹でてお湯を捨てる）と、カリウムを減らせます
- 果物は1日に握りこぶし1個分までが目安です

### よくある質問

**Q: お水はたくさん飲んだほうがいいですか？**

A: 特に制限はありませんが、のどが渇いたときに普通に飲む程度で大丈夫です。担当の先生に「1日どのくらい」と聞いておくと安心です。

**Q: 外食やお惣菜は食べてもいいですか？**

A: 食べても大丈夫ですが、味つけが濃いものが多いので、汁物を残す、お醤油を控える等の工夫をしてください。

**Q: 好きな食べ物を全部やめないといけませんか？**

A: 全部やめる必要はありません。量を少し調整するだけで十分です。無理なく続けることが一番大切です。

### こんな時はすぐ相談してください

- 足や顔がいつもよりむくんでいるとき
- おしっこの量が急に減ったとき
- 体がだるくて食欲がないとき
- 息苦しさを感じるとき

## 活用のポイント

- **対象者を具体的に指定する**: 「80代の高齢者」と「30代の会社員」では、適切な表現や例えが大きく異なります。年齢・職業・理解度を明記してください。
- **たとえ話を活用する**: AIに「身近なたとえを使って」と指示すると、腎臓を「フィルター」、血管を「水道管」のように例えた説明を生成します。
- **やるべきことを具体的に示す**: 「塩分を控える」ではなく「お味噌汁は1日1杯まで」のように、行動レベルで示すと実践しやすくなります。
- **不安を煽りすぎない**: 「放置すると透析になります」のような表現は避け、「今の段階で気をつければ、腎臓を長持ちさせることができます」のように前向きな表現を心がけてください。
- **印刷用に整形する**: 生成された資料を印刷して患者に渡す場合、文字サイズは14pt以上を推奨します（高齢者向け）。
