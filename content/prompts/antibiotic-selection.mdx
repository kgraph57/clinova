# 抗菌薬選択支援

## 概要

経験的抗菌薬治療の選択は、感染巣の推定、起炎菌の予測、患者背景（腎機能、アレルギー）を総合的に判断する必要があります。

## 安全に関する重要な注意事項

抗菌薬の最終選択は感染症専門医またはAST（抗菌薬適正使用支援チーム）と協議してください。AIの出力は施設のアンチバイオグラムを反映していません。

## プロンプトテンプレート

```text
あなたは感染症専門医です。以下の患者情報に基づき、経験的抗菌薬治療の推奨を提示してください。

# 患者情報
- 年齢: {{年齢}}歳、体重: {{体重}}kg
- 推定感染巣: {{感染巣}}（例: 肺炎、尿路感染、腹腔内感染）
- 市中感染/院内感染: {{市中/院内}}
- 重症度: {{軽症/中等症/重症/敗血症性ショック}}
- 培養結果: {{未着/グラム染色所見/培養結果}}

# 患者背景
- 腎機能: eGFR {{eGFR}} mL/min、Cr {{Cr}} mg/dL
- 肝機能: AST {{AST}}, ALT {{ALT}}
- アレルギー: {{薬剤アレルギー（特にペニシリン系）}}
- 免疫状態: {{正常/免疫不全（原因:___）}}
- 過去3か月の抗菌薬使用歴: {{あれば記載}}
- デバイス: {{CV、尿カテ、人工関節等}}

# 出力形式
1. **推定起炎菌**: 感染巣と臨床情報から推定される主な起炎菌
2. **推奨抗菌薬**: 第一選択と代替薬（アレルギー時）
   - 薬剤名、用量（腎機能調整済み）、投与間隔、投与経路
3. **De-escalation計画**: 培養結果判明後の狭域化の方針
4. **治療期間**: ガイドラインに基づく推奨治療期間
5. **モニタリング**: TDM対象薬、腎機能フォロー、効果判定の時期

注意：施設のアンチバイオグラムを参照してください。最終判断は感染症専門医と協議を。
```
