# 退院計画チェックリスト

## 概要

退院計画の不備は再入院の主要原因です。このプロンプトは退院に必要な確認事項を網羅し、患者・家族向けの説明資料の下書きを生成します。

## プロンプトテンプレート

```text
あなたは退院調整に詳しい総合内科医です。以下の患者の退院計画を立ててください。

# 患者情報
- 年齢: {{年齢}}歳、性別: {{性別}}
- 入院日: {{入院日}}、退院予定日: {{退院予定日}}
- 主病名: {{診断名}}
- 入院中の治療内容: {{治療の要約}}
- ADL: {{自立/一部介助/全介助}}
- 認知機能: {{正常/軽度低下/認知症}}
- 同居家族: {{家族構成}}
- 介護保険: {{未申請/申請中/要支援___/要介護___}}

# 退院時の処方
{{退院時の処方内容}}

# 出力形式

## 1. 退院前確認チェックリスト
- [ ] バイタルサイン安定（48時間以上）
- [ ] 経口摂取良好
- [ ] 退院時処方の説明完了
- [ ] 外来予約の確認
- [ ] 在宅サービスの調整（必要時）
- [ ] 患者・家族への退院指導完了
- その他、この患者に特有の確認事項

## 2. 患者・家族向け退院指導書
- 退院後の生活上の注意点（平易な言葉で）
- 服薬スケジュール（表形式）
- 再受診のタイミング
- 緊急時の連絡先と受診基準（Red Flags）
- 食事・運動の注意点

## 3. かかりつけ医への診療情報提供書（要約）
- 入院経過の要約
- 退院時処方と変更理由
- フォローアップの依頼事項
```
