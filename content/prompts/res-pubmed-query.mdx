# PubMed検索式ビルダー

## 概要

PICOに基づき、PubMedで効率的に文献を検索するための検索式（MeSH termsを含む）を作成します。

## プロンプトテンプレート

```text
以下の研究テーマ（PICO）に基づいて、PubMedで使用するための最適な検索クエリを作成してください。
適切なMeSH Termsとキーワードを組み合わせ、論理演算子（AND, OR）を正しく使用してください。

# 研究テーマ (PICO)
[pico]

# 出力
1. **検索クエリ**: コピーしてそのまま使える検索式
2. **解説**: 使用したMeSH Termsとその選定理由
```
