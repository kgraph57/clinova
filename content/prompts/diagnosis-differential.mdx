# 鑑別診断ジェネレーター

## 概要

このプロンプトは、患者の主訴・現病歴・バイタルサインなどの臨床情報をもとに、体系的な鑑別診断リストを生成します。Critical（見逃してはいけない疾患）、Common（頻度の高い疾患）、Rare（稀だが考慮すべき疾患）の3段階で整理することで、診断の漏れを防ぎます。

## 安全に関する重要な注意事項

このプロンプトは診断の補助ツールです。AIの出力は参考情報であり、最終的な診断は必ず医師の臨床判断に基づいて行ってください。入力情報の不足や誤りがあると、重要な疾患を見落とす可能性があります。AIが提示する鑑別診断はすべてを網羅するものではなく、臨床所見・検査結果と合わせた総合的な判断が不可欠です。

## プロンプトテンプレート

```text
あなたは熟練した総合診療医です。以下の症例情報に基づき、鑑別診断リストを作成してください。

# 症例情報
- 主訴: [主訴を入力]
- 現病歴: [現病歴を入力]
- 既往歴: [既往歴を入力]
- バイタルサイン: 体温__°C、血圧__/__mmHg、脈拍__/分、呼吸数__/分、SpO2__%

# 出力形式
1. **Critical（見逃してはいけない疾患）**: 3つ
2. **Common（頻度の高い疾患）**: 3つ
3. **Rare（稀だが考慮すべき疾患）**: 2つ

各疾患について、この症例で疑う根拠と、除外するために必要な追加検査を簡潔に記載してください。

注意：この出力は診断の補助であり、最終判断は必ず医師が臨床所見と合わせて行ってください。
```

## 使い方

1. **主訴を入力**: 患者が最も訴えている症状を記載します（例：発熱、胸痛、腹痛）。
2. **現病歴を入力**: 発症からの経過、増悪・寛解因子、随伴症状などを時系列で記載します。
3. **既往歴を入力**: 既往疾患、服薬歴、アレルギー歴を記載します。
4. **バイタルサインを入力**: 来院時のバイタルサインを数値で記載します。

## 使用例

以下は実際の入力例です。

```text
- 主訴: 発熱
- 現病歴: 3日前から38.5°Cの発熱あり。昨日から咳嗽も出現。
  解熱剤を内服するも改善せず。食欲低下あり。
- 既往歴: 高血圧（アムロジピン5mg内服中）
- バイタルサイン: 体温38.8°C、血圧130/85mmHg、脈拍98/分、
  呼吸数22/分、SpO2 95%
```

この入力に対し、AIは以下のような鑑別診断を提示します。

- **Critical**: 肺炎（市中肺炎）、敗血症、結核
- **Common**: インフルエンザ、急性気管支炎、COVID-19
- **Rare**: 肺塞栓症、悪性リンパ腫

各疾患について、疑う根拠と追加検査（胸部X線、血液培養、喀痰培養、インフルエンザ迅速検査など）が提案されます。

## 活用のポイント

- 入力情報はできるだけ詳細に記載すると、より精度の高い鑑別診断が得られます。
- バイタルサインの異常値は診断の手がかりとなるため、正確に入力してください。
- AIの出力を鵜呑みにせず、臨床所見や画像所見と照らし合わせて検証してください。
- 特にCriticalに分類された疾患は、優先的に除外診断を行うことを推奨します。
