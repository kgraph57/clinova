# キャリアアップ研修会 — AI活用の最初の一歩

> 「AIは難しくない。特別なスキルがなくても使える。明日から始められる最初の一歩。」——300名を超える全職員に、この一言を届けるために設計した60分間。

---

## セミナー概要

| 項目 | 内容 |
|------|------|
| 日時 | 2026年1月22日（木）12:00〜13:00 |
| 時間 | 60分 |
| 場所 | 研究所 セミナールーム |
| 対象 | 全職員＋外部（医療職＋事務職員） |
| 形式 | 実地＋オンラインのハイブリッド開催 |
| 参加者 | 300名超 |
| 講師 | 岡本 賢（小児科専攻医 教育研修センター） |
| 配信 | 開催1週間後、Wisdomebaseにて配信 |

## 核となるメッセージ

> **「AIは難しくない。特別なスキルがなくても使える。明日から始められる最初の一歩」**

ChatGPTをはじめとする生成AIは、特別なスキルがなくても誰でも使える便利なツールです。
「AIって難しそう」「自分には関係ない」と思っている方にこそ聞いていただきたい内容をお届けしました。

## セミナーの目標

| 目標 | 内容 |
|------|------|
| 認知 | 「AIって難しそう」→「使ってみたい」に変える |
| 実践 | 「帰ったら1つ試してみよう」という具体的なアクションを起こす |
| 安全 | リスクを理解し、正しく使えるようになる |
| ゆとり | 日々の業務負担を少しだけ減らし、心にゆとりを持って働く |

## 実演内容

当日は、実際の画面を操作しながら、職種を問わず明日から役立つ活用例を実演しました。

- **分からないことをすぐ調べる** — 検索では見つけにくい情報をAIに質問
- **記録やメールの下書きを作る** — 文章作成の時間を大幅に短縮
- **長い資料を要約する** — 大量の文書から要点を抽出
- **翻訳と英語対応** — 英語の論文やメールをスムーズに処理

## 対象者に合わせた設計

- **AI初心者（ほとんど触ったことがない人）** を中心に設計
- **「自分には関係ない」と思っている人** にこそ届く構成
- **医療職から事務職まで** 職種を問わない汎用的な内容
- **デモ重視**: 難しい専門知識は一切不要、実際の画面を見せて理解を促進

## 成果

300名を超える参加者を集め、ハイブリッド形式での大規模研修会として成功。
後日のWisdomebase配信により、当日参加できなかった職員にも内容を届けることができました。
