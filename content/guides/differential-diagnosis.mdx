# 鑑別診断リスト生成：AIで見落としを防ぐ

## はじめに

臨床現場で最も重要なスキルの一つが、網羅的な鑑別診断を挙げる能力です。
しかし、経験の浅い研修医にとって、主訴から全ての可能性を思い浮かべるのは容易ではありません。

「見落としてはいけない疾患（Don't miss diagnosis）を見逃してしまった...」

このような経験は、多くの研修医が抱える悩みです。
このガイドでは、AIを活用して網羅的な鑑別診断リストを生成する方法を紹介します。

---

## なぜAI支援が必要なのか？

### 鑑別診断の課題

経験不足として、教科書的な知識はあっても、実際の症例で思い出せないという課題があります。
時間的制約として、救急外来では、ゆっくり考える時間がないという課題があります。
認知バイアスとして、最初に思いついた診断に固執してしまう（アンカリングバイアス）という課題があります。
稀な疾患として、頻度の低い疾患は思い浮かばないという課題があります。

### AI支援の効果

AIを活用することで、経験に依存せず、網羅的なリスト生成が可能になります。
従来の方法では時間がかかりますが、AI支援では数秒で生成できます。
従来の方法ではバイアスがかかりますが、AI支援では客観的な提案が可能です。
従来の方法では稀な疾患を見落としますが、AI支援では稀な疾患も含めることができます。

# ステップ1: 鑑別診断の基本原則を理解する

## 鑑別診断とは？

鑑別診断とは、患者の症状や所見から考えられる疾患を列挙し、優先順位をつけて診断を絞り込むプロセスです。

---

## 鑑別診断の基本原則

### 1. VINDICATE法

鑑別診断を網羅的に考えるためのフレームワークです。

| 頭文字 | カテゴリー | 例 |
| :---: | :--- | :--- |
| V | Vascular（血管性） | 脳梗塞、心筋梗塞、大動脈解離 |
| I | Inflammatory/Infectious（炎症・感染） | 肺炎、髄膜炎、心内膜炎 |
| N | Neoplastic（腫瘍） | 肺癌、脳腫瘍、悪性リンパ腫 |
| D | Degenerative/Drugs（変性・薬剤） | 変形性関節症、薬剤性肝障害 |
| I | Iatrogenic/Idiopathic（医原性・特発性） | 術後合併症、特発性血小板減少性紫斑病 |
| C | Congenital（先天性） | 先天性心疾患、遺伝性疾患 |
| A | Autoimmune/Allergic（自己免疫・アレルギー） | 関節リウマチ、SLE、アナフィラキシー |
| T | Trauma（外傷） | 骨折、頭部外傷 |
| E | Endocrine/Metabolic（内分泌・代謝） | 糖尿病、甲状腺機能亢進症 |

---

### 2. Common things are common

頻度の高い疾患を優先的に考える

> 「馬の蹄の音が聞こえたら、シマウマではなく馬を考えよ」

#### 例: 発熱の鑑別診断

頻度の高い疾患:
- 上気道炎（風邪）
- インフルエンザ
- 尿路感染症
- 肺炎

頻度の低い疾患:
- 心内膜炎
- 結核
- 悪性腫瘍

→ まず頻度の高い疾患を除外してから、稀な疾患を考える

---

### 3. Don't miss diagnosis（見逃してはいけない疾患）

致死的または緊急治療が必要な疾患を優先的に考える

#### 主訴別のDon't miss diagnosis

| 主訴 | Don't miss diagnosis |
| :--- | :--- |
| 胸痛 | 急性冠症候群、大動脈解離、肺塞栓症、気胸 |
| 呼吸困難 | 肺塞栓症、急性心不全、気胸、アナフィラキシー |
| 腹痛 | 腸管穿孔、腸管虚血、大動脈瘤破裂、異所性妊娠 |
| 頭痛 | くも膜下出血、髄膜炎、脳腫瘍、側頭動脈炎 |
| 意識障害 | 低血糖、脳卒中、髄膜炎、中毒 |

---

### 4. 優先順位付けの3つの基準

鑑別診断をリストアップしたら、以下の基準で優先順位をつけます。

#### 基準1: 緊急性（Urgency）

致死的または緊急治療が必要か？

- 高: 大動脈解離、肺塞栓症、くも膜下出血
- 中: 肺炎、尿路感染症
- 低: 風邪、筋肉痛

---

#### 基準2: 頻度（Frequency）

どのくらい頻度が高いか？

- 高頻度: 風邪、胃腸炎、筋肉痛
- 中頻度: 肺炎、尿路感染症
- 低頻度: 心内膜炎、結核、悪性腫瘍

---

#### 基準3: 所見の一致度（Fit）

患者の症状・所見とどのくらい一致するか？

- 高一致: 支持する所見が多い
- 中一致: 一部の所見が一致
- 低一致: 否定的な所見が多い

---

## 優先順位付けのマトリックス

| 診断名 | 緊急性 | 頻度 | 一致度 | 優先順位 |
| :--- | :---: | :---: | :---: | :---: |
| 急性冠症候群 | 高 | 中 | 高 | ★★★ |
| 大動脈解離 | 高 | 低 | 中 | ★★★ |
| 肺塞栓症 | 高 | 低 | 中 | ★★★ |
| 肺炎 | 中 | 高 | 高 | ★★ |
| 胃食道逆流症 | 低 | 高 | 中 | ★ |

## 鑑別診断の落とし穴

### 落とし穴1: アンカリングバイアス

最初に思いついた診断に固執してしまう

例:
- 胸痛 → 急性冠症候群と決めつける
- 心電図正常 → 「急性冠症候群ではない」と判断
- 大動脈解離を見逃す

対策:
- 必ず複数の鑑別診断を挙げる
- 否定的な所見があっても、他の鑑別を考える

---

### 落とし穴2: 確証バイアス

自分の仮説を支持する情報ばかり集めてしまう

例:
- 「肺炎だろう」と思い込む
- 発熱、咳、肺野の浸潤影に注目
- 肺塞栓症の可能性を見落とす

対策:
- 否定的な所見も積極的に探す
- 「この診断を否定する所見は何か？」と自問する

---

### 落とし穴3: 可用性バイアス

最近経験した症例や印象的な症例を優先してしまう

例:
- 最近、大動脈解離の症例を経験
- 胸痛患者を見ると、すぐに大動脈解離を疑う
- 頻度の高い急性冠症候群を見落とす

対策:
- 頻度を考慮する（Common things are common）
- 稀な疾患を除外する前に、頻度の高い疾患を除外

```
- 主訴: [ここに入力]
- 重要な所見: [ここに入力]

【出力形式】
以下の形式で鑑別診断をリストアップしてください：

1. 診断名

   - 否定的な所見:
   - 確定診断の方法:
   - 緊急性: 高/中/低

最低5つ、最大10個の鑑別診断を提示してください。
```

### AIの出力例

```
【鑑別診断リスト】

1. 急性冠症候群（ACS） ★
   - 支持する所見: 締め付けられる胸痛、放散痛、冷や汗、喫煙歴、高血圧、脂質異常症
   - 否定的な所見: なし
   - 確定診断の方法: 心電図、心筋逸脱酵素（トロポニン）、冠動脈造影
   - 緊急性: 高

2. 大動脈解離
   - 支持する所見: 突然発症の胸痛
   - 否定的な所見: 背部痛なし、血圧左右差なし（確認必要）
   - 確定診断の方法: 造影CT、経食道心エコー
   - 緊急性: 高

   - 支持する所見: 突然発症の胸痛
   - 否定的な所見: 呼吸困難なし、下肢腫脹なし（確認必要）
   - 確定診断の方法: D-dimer、造影CT
   - 緊急性: 高

4. 胃食道逆流症（GERD）
   - 支持する所見: 胸痛
   - 否定的な所見: 食事との関連なし、胸焼けなし
   - 確定診断の方法: 上部消化管内視鏡、プロトンポンプ阻害薬（PPI）テスト
   - 緊急性: 低

5. 筋骨格系の痛み（肋間神経痛など）
   - 支持する所見: 胸痛
   - 否定的な所見: 圧痛なし（確認必要）、体動との関連なし
   - 確定診断の方法: 身体所見、除外診断
   - 緊急性: 低
```

# ステップ2: 鑑別診断の優先順位付け

## 優先順位付けの重要性

AIが生成した鑑別診断リストは、網羅的である一方、玉石混交です。臨床的に重要な疾患を見逃さないために、優先順位付けが不可欠です。

### 優先順位付けの3つの基準

1.  緊急性 (Urgency): 見逃すと生命に関わるか？
2.  頻度 (Frequency): よくある疾患か？ (Common things are common)
3.  所見の一致度 (Fit): 患者の所見とどれだけ一致するか？

---

## AIを使った優先順位付け

AIに優先順位付けをさせることで、客観的かつ迅速にリストを整理できます。

### 優先順位付けのプロンプト

```
あなたは経験豊富な臨床医です。以下の鑑別診断リストを、3つの基準（緊急性、頻度、所見の一致度）に基づいて優先順位付けしてください。

【鑑別診断リスト】
[ここにAIが生成したリストを貼り付け]

【出力形式】
以下の表形式で整理してください：

| 診断名 | 緊急性 (高/中/低) | 頻度 (高/中/低) | 所見の一致度 (高/中/低) | 総合優先度 (1-5位) |
| :--- | :---: | :---: | :---: | :---: |

最も可能性の高い診断を★で示してください。
```

## 思考プロセスを可視化する

AIに優先順位付けの思考プロセスを説明させることで、臨床推論の学習にも役立ちます。

### 思考プロセス説明のプロンプト

```
上記の優先順位付けについて、あなたの思考プロセスを説明してください。なぜその順位になったのか、各疾患を比較しながら解説してください。
```

### AIの出力例

```
【優先順位付けの思考プロセス】

今回の症例は「60歳男性、リスク因子多数、突然発症の典型的な胸痛」です。この情報から、まず生命を脅かす疾患を最優先に考えます。

1.  急性冠症候群（ACS）を1位とした理由:
    *   所見の一致度が最も高い: 締め付けられる胸痛、放散痛、冷や汗、そして多数のリスク因子（年齢、性別、喫煙、高血圧、脂質異常症）は、典型的なACSの症状です。
    *   緊急性が高い: 見逃せば心筋梗塞に至り、生命に関わります。
    *   頻度も中程度: 胸痛の原因として決して稀ではありません。
    以上の理由から、ACSを最も疑い、迅速な心電図・心筋逸脱酵素の確認が必要です。

2.  大動脈解離を2位とした理由:
    *   緊急性が非常に高い: ACS同様、見逃しは致死的です。
    *   所見の一部が一致: 「突然発症の胸痛」という点は一致します。
    *   頻度が低い: ACSと比較すると頻度は低いです。また、典型的な背部痛や血圧左右差の記載がないため、ACSよりは優先度を下げました。しかし、常に念頭に置くべき疾患です。

3.  肺塞栓症（PE）を3位とした理由:
    *   緊急性が高い: これも見逃しは致死的です。
    *   所見の一致度が低い: 「突然発症の胸痛」は一致しますが、呼吸困難や下肢腫脹といったPEを示唆する典型的な所見がありません。そのため、大動脈解離よりもさらに優先度を下げました。

4.  GERDと筋骨格系の痛みを下位とした理由:
    *   緊急性が低い: 生命を脅かす可能性は低いです。
    *   所見の一致度が低い: 症状は非典型的です。
    *   除外診断: これらは、上記の致死的疾患を否定した後に考慮すべき診断です。

結論として、 この症例では、まずACSを最優先に考え、同時に大動脈解離とPEの可能性も念頭に置きながら、検査を進めるべきです。
```

✅ 鑑別診断の優先順位付けの3つの基準（緊急性、頻度、所見の一致度）
✅ AIを使って客観的に優先順位付けを行うプロンプト
✅ 優先順位付けの思考プロセスをAIに説明させる方法

# ステップ3: 除外診断の根拠を明確にする

## 除外診断の重要性

鑑別診断のプロセスでは、「何を疑うか」と同時に「なぜそれを除外するのか」を明確にすることが極めて重要です。これにより、臨床推論の論理性が担保され、カンファレンスやカルテ記載において、他者への説明責任を果たすことができます。

AIを活用して、この「除外の根拠」を言語化するトレーニングを行いましょう。

---

## 除外根拠を明確にするプロンプト

AIに、優先度の低い診断を除外する理由を説明させるプロンプトです。

### 基本プロンプト

```
あなたは経験豊富な臨床医です。以下の症例と鑑別診断リストについて、優先度の低い診断（3位以下）を除外する根拠を説明してください。

【症例情報】
[ここに症例情報を貼り付け]

【鑑別診断リスト（優先順位付き）】
[ここに優先順位付けしたリストを貼り付け]

【出力形式】
以下の形式で、各診断の除外根拠を説明してください：

診断名: [診断名]
*   支持する所見（なぜ鑑別に挙がったか）:
*   否定的な所見（なぜ優先度が低いか）:
*   追加で確認すべき項目:
```

```
1. 急性冠症候群（ACS） ★
2. 大動脈解離
3. 肺塞栓症（PE）
4. 胃食道逆流症（GERD）
5. 筋骨格系の痛み

*   支持する所見（なぜ鑑別に挙がったか）:
*   否定的な所見（なぜ優先度が低いか）:
*   追加で確認すべき項目:
```

### AIの出力例

```
【除外診断の根拠】

診断名: 胃食道逆流症（GERD）
*   支持する所見: 胸痛という症状そのもの。非心臓性の胸痛の原因として最も頻度が高い疾患の一つであるため、鑑別には必ず挙がります。
*   否定的な所見: 本症例の胸痛は「突然発症」「締め付けられる」「放散痛」「冷や汗」といった虚血性心疾患を強く示唆する性状です。GERDに典型的な「食後の増悪」「胸焼け」「酸っぱいものが上がってくる感じ」といった症状がありません。そのため、優先度は低いと判断します。
*   追加で確認すべき項目: 食事との関連、胸焼けの有無、PPI（プロトンポンプ阻害薬）のトライアル投与への反応。

診断名: 筋骨格系の痛み（肋間神経痛など）
*   支持する所見: 胸痛という症状。これも非心臓性胸痛として頻度が高いです。
*   否定的な所見: 筋骨格系の痛みは、通常「体動時痛」や「圧痛」を伴います。本症例ではこれらの記載がなく、むしろ安静時にも持続する内臓痛パターンです。そのため、可能性は低いと考えます。
*   追加で確認すべき項目: 胸壁の圧痛の有無、深呼吸や体動による痛みの変化。
```

## 「もし〜だったら？」で思考を深める

臨床推論のトレーニングとして、「もし所見が少し違ったら、鑑別診断はどう変わるか？」とAIに問いかけることは非常に有効です。

### 

### 「もし〜だったら？」思考実験プロンプト

```
あなたは経験豊富な臨床医です。先ほどの胸痛の症例について、以下の「もし〜だったら？」という条件下で、鑑別診断の優先順位がどのように変わるか説明してください。

【条件1】もし、胸痛が「引き裂かれるような背部痛」で、「血圧に左右差」があったら？
【条件2】もし、胸痛に加えて「呼吸困難」と「右下肢の腫脹」があったら？
【条件3】もし、胸痛が「食後に増悪」し、「胸焼け」を伴っていたら？
```

---

### AIの出力例

```
【「もし〜だったら？」思考実験】

【条件1】もし、胸痛が「引き裂かれるような背部痛」で、「血圧に左右差」があったら？

*   鑑別診断の変化: 大動脈解離が最優先（1位）になります。
*   理由: 「引き裂かれるような背部痛」と「血圧の左右差」は、大動脈解離に極めて典型的な所見です。ACSの可能性も残りますが、これらの所見があれば、まず大動脈解離を否定するための造影CTを最優先で考慮すべきです。

【条件2】もし、胸痛に加えて「呼吸困難」と「右下肢の腫脹」があったら？

*   鑑別診断の変化: 肺塞栓症（PE）の優先順位がACSと同等、あるいはそれ以上に上がります。
*   理由: 「呼吸困難」と「下肢腫脹（DVTを示唆）」は、PEの古典的な三徴（胸痛、呼吸困難、喀血）のうち2つを満たし、リスクを著しく高めます。Wells' scoreも上昇し、D-dimerや造影CTの必要性が高まります。

【条件3】もし、胸痛が「食後に増悪」し、「胸焼け」を伴っていたら？

*   鑑別診断の変化: 胃食道逆流症（GERD）の優先順位が上がります。
*   理由: 「食後の増悪」と「胸焼け」は、GERDに典型的な症状です。ただし、ACSは非典型的な症状で発症することもあるため、リスク因子を持つ本症例では、まず心電図などでACSを否定することが重要です。致死的疾患を除外した上で、GERDの治療（PPIテストなど）を検討します。
```

# ステップ1: 複雑な症例での鑑別診断

## 複雑な症例とは？

実臨床では、単一の主訴だけでなく、複数の問題が絡み合った複雑な症例に遭遇します。

*   複数の主訴: 例）発熱と腹痛と意識障害
*   複数の既往歴: 例）糖尿病、腎不全、心不全を合併
*   非典型的な症状: 教科書通りではない症状

このような場合、鑑別診断は困難を極めます。AIを壁打ち相手として活用し、思考を整理しましょう。

---

## プロブレムリストを作成する

まず、症例の情報をプロブレムリストとして整理します。これは、診断の手がかりとなる陽性所見や異常値をリストアップする作業です。

### プロブレムリスト作成のプロンプト

```
あなたは経験豊富な臨床医です。以下の症例情報から、プロブレムリストを作成してください。

【症例情報】
[ここに症例情報を貼り付け]

【出力形式】
重要な順にプロブレムをリストアップし、それぞれに簡単な解説を加えてください。
```

### 実践例: 発熱と意識障害の症例

症例情報:
> 75歳男性。3日前からの発熱と食欲不振を主訴に来院。来院時、JCS II-10と意識障害を認める。既往に糖尿病、高血圧、2年前に前立腺癌で手術歴あり。バイタルは体温38.5℃、血圧90/50mmHg、脈拍110/分、SpO2 92%。身体所見で項部硬直はなし。血液検査でWBC 18,000, CRP 25, Cre 2.5。尿検査で混濁あり。

入力プロンプト:
```
あなたは経験豊富な臨床医です。以下の症例情報から、プロブレムリストを作成してください。

【症例情報】
75歳男性。3日前からの発熱と食欲不振を主訴に来院。来院時、JCS II-10と意識障害を認める。既往に糖尿病、高血圧、2年前に前立腺癌で手術歴あり。バイタルは体温38.5℃、血圧90/50mmHg、脈拍110/分、SpO2 92%。身体所見で項部硬直はなし。血液検査でWBC 18,000, CRP 25, Cre 2.5。尿検査で混濁あり。

```

AIの出力例:
```
【プロブレムリスト】

#1. 敗血症性ショック (Sepsis with Shock)
   - 解説: 発熱、頻脈、低血圧、意識障害、高炎症反応から、SIRS基準を満たし、臓器障害（意識障害、腎機能障害）を伴う敗血症であり、循環不全（ショック）を合併している状態です。最も緊急性の高い問題です。

#2. 意識障害 (Disturbance of Consciousness)
   - 解説: JCS II-10。敗血症による脳症、中枢神経系感染症、代謝性脳症など、多岐にわたる原因を考慮する必要があります。

#3. 急性腎障害 (Acute Kidney Injury, AKI)
   - 解説: Cre 2.5。敗血症による腎前性腎不全が最も考えられますが、薬剤性や腎後性も鑑別が必要です。

#4. 感染巣 (Infection Focus)
   - 解説: 敗血症の原因となっている感染巣の特定が急務です。尿混濁があることから尿路感染症が最も疑われますが、肺炎、胆管炎、カテーテル関連血流感染症なども鑑別します。

#5. 糖尿病 (Diabetes Mellitus)
   - 解説: 感染症の易感染性や、高血糖による意識障害（HHSなど）のリスク因子となります。

#6. 前立腺癌術後 (Post-prostatectomy state)
   - 解説: 尿路の構造変化により、複雑性尿路感染症のリスク因子となります。
```

```
【プロブレムリスト】
[ここにプロブレムリストを貼り付け]

【出力形式】
診断名と、その診断が各プロブレムをどのように説明できるかを記述してください。
```

### AIの出力例

```
【統合的鑑別診断】

1.  尿路性敗血症 (Urosepsis) ★
    *   説明: 尿混濁・前立腺癌術後という背景から尿路感染症が先行し、菌血症から敗血症性ショックに至ったと考えれば、#1, #2, #3, #4, #5, #6の全てのプロブレムを合理的に説明できます。最も可能性の高い診断です。

2.  肺炎 (Pneumonia) に伴う敗血症
    *   説明: 高齢者・糖尿病患者は肺炎のリスクが高く、肺炎から敗血症性ショックに至ることも多いです。ただし、呼吸器症状の記載がないため、尿路感染症よりは優先度が下がります。

3.  胆管炎 (Cholangitis) に伴う敗血症
    *   説明: 胆管炎も敗血症の原因として重要です。しかし、腹痛や黄疸の記載がないため、可能性は低いと考えられます。

4.  髄膜炎 (Meningitis)
    *   説明: 発熱と意識障害をきたす疾患として重要です。しかし、項部硬直がないこと、他の臓器障害（腎障害）を説明しにくいことから、優先度は低いです。

5.  感染性心内膜炎 (Infective Endocarditis)
    *   説明: 亜急性の経過で発熱、塞栓による脳梗塞（意識障害）、免疫複合体による腎障害をきたす可能性があり、鑑別には挙がります。しかし、心雑音や塞栓を示唆する皮膚所見がないため、積極的には疑いにくいです。
```

# ステップ2: 鑑別診断のマインドマップを作成する

## なぜマインドマップなのか？

鑑別診断のリストは、テキストだけでは全体像を把握しにくいことがあります。マインドマップを使って思考プロセスを可視化することで、以下のメリットがあります。

*   全体像の把握: 疾患同士の関連性や分類が一目でわかります。
*   思考の整理: 複雑な鑑別診断を構造的に整理できます。
*   記憶の定着: 文字情報よりも視覚的な情報の方が記憶に残りやすいです。
*   プレゼンテーション: カンファレンスなどで、自身の思考プロセスを他者に分かりやすく伝えられます。

---

## AIによるマインドマップ生成

AIに鑑別診断リストをマインドマップ形式で出力させることで、簡単に思考を可視化できます。

### マインドマップ生成プロンプト

```
あなたは経験豊富な臨床医です。以下の主訴と鑑別診断リストについて、思考プロセスが分かるようにマインドマップ形式で整理してください。

【主訴】
[ここに主訴を入力]

【鑑別診断リスト】
[ここに鑑別診断リストを貼り付け]

【出力形式】
*   主訴を中心に配置してください。

*   各診断名には、診断のキーワードとなる所見を3つ程度追記してください。

```
