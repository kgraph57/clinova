# 診断書・紹介状作成支援：AIで医療文書作成を効率化・標準化

## はじめに

診断書、紹介状、退院サマリーといった医療文書の作成は、医師の日常業務において大きな割合を占める、重要でありながら時間のかかる作業です。
これらの文書は、患者の法的な権利を守り、施設間の連携をスムーズにし、継続的なケアの質を担保する上で不可欠な役割を担っています。

しかし、多忙な臨床業務の中で、これらの文書を正確に、分かりやすく作成することは、多くの医師にとって負担となっています。
「何を書けばいいのか迷う」「定型的な文章の入力に時間がかかる」「記載内容の抜け漏れが心配」といった悩みは、誰もが経験するものです。

このガイドでは、AIを活用して、診断書や紹介状などの医療文書作成を劇的に効率化し、質を向上させる方法を紹介します。
AIを優秀なメディカルライターとして活用し、文書作成の負担を軽減することで、より多くの時間を患者と向き合うために使いましょう。

## なぜAI支援が必要なのか？

### 医療文書作成の課題

診療時間外に多くの時間を費やしているという時間的負担があります。
医師によって記載内容の詳しさや形式が異なり、受け取り手が混乱することがあります。
記載すべき重要な情報、例えばアレルギー歴や主要な現病歴などが欠落する可能性があります。
簡潔かつ的確に、必要な情報を過不足なく伝える文章を作成するのは難しいものです。
特に、休職診断書や法的文書など、患者の人生に大きな影響を与える文書作成は精神的な負担が大きいです。

### AI支援の効果

AIを活用することで、これらの課題を克服し、文書作成の質と効率を向上させることができます。

患者情報や目的を指示するだけで、AIが文書のドラフト（下書き）を数秒で作成します。
目的に応じて、紹介、返信、公的機関への提出など、適切なフォーマットで構造化された文書を作成します。
記載すべき項目を網羅したテンプレートをAIが提示することで、重要な情報の欠落を防ぎます。
専門的で、誰が読んでも分かりやすい、洗練された文章を生成します。
患者の背景、例えば年齢、職業、社会的背景などを考慮した、個別性の高い文書作成を支援します。

## 非常に重要な注意事項

AIが生成した文書は、あくまで「下書き（ドラフト）」です。
最終的な内容の正確性、妥当性、法的な責任は、すべて署名する医師にあります。

AIの生成結果を、絶対にそのままコピー＆ペーストして使用しないでください。

その患者固有の状況や背景が、適切に反映されているか、個別性を確認してください。
AIが生成した文章のニュアンスが、伝えたい意図と合っているか、ニュアンスを確認してください。

AIは強力なツールですが、万能ではありません。
最終的な責任は、プロフェッショナルであるあなた自身にあることを、常に心に留めておいてください。

# ステップ1: 診断書のドラフトを生成する

## 診断書の基本構成

診断書は、患者の健康状態について、医師が専門的な見地から証明する公的な文書です。
その目的は、休職、保険請求、公的支援の申請など多岐にわたります。
どのような目的であれ、診断書には共通して記載すべき基本要素があります。

まず、患者情報として、氏名、生年月日、性別、住所を記載します。
次に、診断名として、主病名と関連する副病名を記載します。
ICD-10コードを併記することもあります。
発症・診断日として、症状が始まった日と診断が確定した日を記載します。
診断の根拠として、症状、身体所見、検査結果など、診断に至った客観的な根拠を記載します。
治療内容と経過として、これまで行ってきた治療、例えば投薬や手術などとその効果を記載します。
現在の状況と今後の見通しとして、現在の症状の程度、治療状況、今後の回復の見込みなどを記載します。
特記事項として、目的に応じて、就労に関する意見、必要な配慮、後遺症の有無などを記載します。
最後に、作成日、病院名、医師名、押印を記載します。

これらの要素を網羅し、目的に応じて記載の詳しさを調整することが重要です。

---

## 診断書作成の基本プロンプト

このプロンプトは、患者の基本情報と診断書の目的をAIに伝えることで、上記の構成要素を網羅した診断書のドラフトを生成させるためのものです。

### 📋 診断書基本プロンプト（コピペ用）

```

【患者情報】
* 氏名: [氏名]
* 生年月日: [生年月日]
* 主病名: [主病名]
* 診断日: [診断日]
* 診断根拠: [症状、所見、検査結果などを簡潔に]
* 治療経過: [これまでの治療内容を簡潔に]
* 現在の状況: [現在の症状や状態を簡潔に]

【診断書の目的】
[例: 会社に提出し、病気療養のための休職を申請する]

【出力形式】
一般的な診断書のフォーマットに従って、丁寧な言葉遣いで記述してください。特に、診断書の目的に関連する「今後の見通し」や「必要な配慮」については、具体的に記述してください。
```

# ステップ2: 紹介状・返信状を作成する

## 医療連携の要、SOAP形式

紹介状（診療情報提供書）やその返信状は、患者の情報を医療機関の間で正確に共有し、継続的で質の高い医療を提供するための、非常に重要なコミュニケーションツールです。

これらの文書を作成する上で、情報を構造化し、抜け漏れなく伝えるための世界標準のフレームワークがSOAP（ソープ）です。

SはSubjective（主観的情報）を表します。
患者の訴え（主訴）、現病歴、既往歴、アレルギー歴などが含まれます。
OはObjective（客観的情報）を表します。
身体所見、バイタルサイン、検査結果、画像所見などが含まれます。
AはAssessment（評価）を表します。
SとOから導き出される、医師による評価、考察、アセスメント、診断が含まれます。
PはPlan（計画）を表します。
Aに基づいて立案される、今後の治療計画、検査計画、紹介の目的などが含まれます。

このSOAP形式を用いることで、受け取った医師は患者の状態を迅速かつ正確に把握することができます。

---

```
あなたは、経験豊富な総合内科医です。以下の患者情報（SOAP形式）に基づき、指定された宛先の専門医への紹介状のドラフトを作成してください。

【宛先】
[例: 〇〇大学病院 呼吸器内科 教授 〇〇先生 御侍史]

【紹介目的】
[例: 精査・加療のため]

【患者情報】
* 患者: [氏名、年齢、性別]
* 紹介元: [あなたの病院・クリニック名、あなたの名前]

【SOAP】
* S (Subjective):
  * 主訴: 
  * 現病歴: 
  * 既往歴: 
  * 生活歴・家族歴: 
  * 内服薬:

  * 身体所見: 
  * 検査データ:

【出力形式】
一般的な紹介状のフォーマットに従い、専門医に対する敬意を払った丁寧な言葉遣いで記述してください。特に、AssessmentとPlanを明確に記述してください。
```

# ステップ1: 退院サマリーを作成する

## 継続的なケアのための情報集約

退院サマリー（退院時要約）は、一人の患者の入院中の情報を集約し、退院後のケアを担当するかかりつけ医、訪問看護師、介護スタッフなど、関係者全員が患者の状態を正確に把握するための、極めて重要な文書です。

質の高い退院サマリーは、退院後の再入院を防ぎ、シームレスな地域連携を実現するための鍵となります。
しかし、入院期間が長くなればなるほど、その情報を簡潔かつ網羅的にまとめるのは大変な作業です。
AIは、この複雑な情報整理と文書化のプロセスを大幅に効率化します。

# ステップ2: 配慮の行き届いた文章表現をAIに相談する

## 言葉の選び方が、患者の未来を左右する

医療文書、特に診断書に書かれる言葉は、単なる事実の記述以上の重みを持ちます。
その一言一句が、患者さんやそのご家族の感情、時にはその後の人生の選択にまで、大きな影響を与えることがあります。

例えば、「治癒の見込みはない」と書くか、「症状のコントロールを目標とします」と書くか。
事実は同じでも、受け手の印象は大きく異なります。

私たちは、医学的な正確性を保ちつつも、患者さんの希望を奪わず、前向きな姿勢を支えるような、温かみのある言葉を選ぶ必要があります。
AIは、このような繊細なニュアンスの表現を考える上での、良き相談相手となります。

---

## 表現を洗練させるためのプロンプト

このプロンプトは、あなたが書いた文章のドラフトをAIに提示し、より患者の心情に配慮した、あるいは、より正確で誤解の少ない表現への言い換えを提案させるためのものです。

### 📋 表現相談プロンプト（コピペ用）

```
あなたは、コミュニケーションの専門家であり、言葉の選び方が患者の心理に与える影響を深く理解しているカウンセラーです。以下の診断書の記述について、より患者さんの心情に配慮した、前向きで温かみのある表現への言い換えを3パターン提案してください。

【元の文章】
[ここに、あなたが書いた、あるいはAIが生成した文章のドラフトを貼り付ける]

【背景・文脈】
* 患者: [患者の年齢、性別、性格など]
* 病名: [病名]
* 状況: [なぜこの記述が必要なのか、その背景]

【求めるニュアンス】
* [例: 希望を失わせず、治療に前向きになれるような表現]
* [例: できないことではなく、できることに焦点を当てた表現]
* [例: 突き放すのではなく、寄り添う姿勢が伝わる表現]
```

## 実践例: 進行がん患者の診断書

### 入力プロンプト

```
あなたは、コミュニケーションの専門家であり、言葉の選び方が患者の心理に与える影響を深く理解している緩和ケア医です。以下の診断書の記述について、より患者さんの心情に配慮した、前向きで温かみのある表現への言い換えを3パターン提案してください。

【元の文章】
「全身に転移を認めるため、根治は困難である。今後は、化学療法による延命と、症状緩和が治療の中心となる。」

【背景・文脈】
* 患者: 65歳男性、前向きな性格だが、最近少し元気をなくしている。
* 病名: 進行性肺がん（ステージIV）
* 状況: 本人への告知後の、生命保険の申請に必要な診断書の記述。

【求めるニュアンス】
* 「根治は困難」という厳しい現実を伝えつつも、絶望感を与えない表現。
* 今後の治療が、単なる「延命」ではなく、QOL（生活の質）を維持するための積極的なものであることが伝わる表現。
```

### AIの出力例

```
承知いたしました。厳しい状況の中でも、患者様が希望を失わず、これからの治療に前向きに取り組めるような表現を提案します。

### 言い換え提案

パターン1（目標の再設定を強調する表現）
> がんが体の広い範囲に見られるため、病気を完全に消し去ることを目指すのではなく、がんと共存しながら、患者様ご自身の『生活の質』を最大限に高めることを、新たな治療目標とします。今後は、化学療法によってがんの進行を穏やかにし、痛みなどの症状を和らげることで、これまでと変わらない穏やかな日常をできるだけ長く維持できるよう、チーム一丸となってサポートして参ります。

パターン2（患者の「時間」を大切にする視点）
> 病気の性質上、完全に治すことは難しい状況ですが、残された貴重な時間を、より豊かで意味のあるものにするための治療が可能です。今後は、化学療法によって病気の進行をコントロールし、症状緩和を積極的に行うことで、ご家族と過ごす時間や、ご自身の趣味を楽しむ時間など、〇〇様にとって大切な時間を守ることを最優先の目標といたします。

パターン3（よりシンプルで、力強い表現）
> 病気は進行しておりますが、治療の選択肢がなくなったわけではありません。これからの治療の主役は、病気を治すことではなく、〇〇様ご自身です。化学療法や緩和ケアを組み合わせることで、体の辛さを和らげ、自分らしい生活を続けることは十分に可能です。私たちは、そのための最善の医療を提供することをお約束します。

【解説】
これらの表現は、いずれも「できないこと（根治）」ではなく「できること（症状緩和、QOL維持）」に焦点を当てています。
「延命」という言葉を避け、「穏やかな日常」「豊かな時間」「自分らしい生活」といった、患者様が具体的にイメージできるポジティブな言葉に置き換えることで、今後の治療に対する前向きな動機付けを促すことを意図しています。
```
